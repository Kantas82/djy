<tr><td VALIGN=TOP width="626"><h2 align=center>演绎十年“文艺复兴” 盘点PC历年经典作</h2>

<h6></h6>
<hr>
	<p>【大纪元2014年07月08日讯】在过去十年中，计算机可以说是历经了一次“文艺复兴”。消费电子产品流行科技网站engadget对此做了一个从2004年起一系列的介绍，那个时候台式机仍然称霸这计算机市场，笔记本电脑的外形相对来讲比较笨重，而平板电脑则仍然是被应用在医学领域中的“利基”产品。</p>
<p>不过，这种状况并没有持续很长的时间。然后，上网本在全球掀起了短暂的热潮，将小型笔记本带入到了大众市场。而Ultrabooks（超级本）的到来则证明，超薄笔记本的功能也同样可以非常强大。然后，在随后的几年中，整个PC市场就不得不面对触摸屏以及诸如智能手机和平板电脑等触摸屏移动设备的兴起。总而言之，在过去的十年之中，计算机技术领域发生了重大的变革。</p>
<p>下面，就让我们一起来看一下过去十年PC领域中的经典代表作：</p>
<p><B>2004年：索尼VAIO X505</B></p>
<p>产品参数：1.1GHz Pentium M处理器，20GB硬盘，重量为1.73磅，分辨率为1024 x 768的10.4英寸显示屏。</p>
<p>当时，索尼并没有意识到VAIO X505笔记本电脑拥有如此巨大的影响力，但是这款设备为接下来10年笔记本电脑的发展打下了坚实的基础。这款10英寸笔记本电脑的机身非常轻薄，非常易于携带，与上网本或超极本很相似。如果索尼不是将VAIO X505的售价定为2999美元那么高，也许它很有可能会开启一个移动计算革命。</p>
<p><B>2005年：IBM ThinkPad T43</B></p>
<p>产品参数：1.6GHz至2.13GHz Pentium M处理器，至少30GB硬盘驱动器，重量为6磅，DVD光驱，分辨率为1024 x 768或1400 x 1050的14.1英寸显示屏。</p>
<p>当时，笔记本电脑还大多只是全球专业人士所使用的一种工具，IBM ThinkPad T43成了计算机时代的绝唱。在联想收购IBM的PC业务之前，ThinkPad T43成为了保留IBM品牌的几款产品之一。ThinkPad T43代表着IBMThinkPad的品牌，运行快速、做工精细，便于携带。</p>
<p><B>2006年：戴尔XPS 700</B></p>
<p>产品参数：Core 2Duo或Core 2 Extreme处理器，双320GB硬盘，双DVD光驱，双GeForce7900GTX显卡。</p>
<p>在2006年以前，戴尔主打高性能的PC设备。如果你想要购买一个大品牌游戏计算机，那么戴尔XPS 700就是一个很好的选择。戴尔XPS700拥有“激进的设计”，并且拥有很强大的处理能力。如果你是资深在线游戏玩家，那么戴尔XPS700必定是你梦寐以求的计算机。</p>
<p><B>2007年：华硕Eee PC 701</B></p>
<p>产品参数：800MHz或900MHz Celeron M处理器，2GB至8GB的固态硬盘，重量为2磅，分辨率为800 x 480的7英寸显示屏。</p>
<p>华硕Eee PC 701的到来标志着上网本热潮的正式来袭，这股热潮一直持续到2010年iPad的到来。华硕Eee PC 701的显示屏、速度和存储空间都很平庸，但是它表明你在咖啡馆里查看邮箱时，根本就不需要一部笨重而又昂贵的笔记本电脑。</p>
<p><B>2008年：苹果MacBookAir</B></p>
<p>产品参数：1.6GHz或1.8GHz Core 2Duo处理器，80GB硬盘或64GB固态硬盘，重量为3磅，分辨率为1280 x 800的13.3英寸显示屏。</p>
<p>苹果MacBookAir是一款典型的超极本，它成功地跨越了便携式笔记本电脑和全功能笔记本电脑之间的界线。苹果MacBookAir足够快速，能够处理复杂的任务，它也非常轻便，让用户很难感觉到它的重量。</p>
<p><B>2009年：惠普Firebird</B></p>
<p>产品参数：2.66GHz或2.83GHz Core 2Quad处理器，双250GB或320GB硬盘，DVD或蓝光光驱，双GeForce9800S显卡。</p>
<p>不管是以任何标准来看，惠普Firebird都算不上是一台完美的设备，但是它却显示了计算机强大运行效率。借助于惠普Firebird，用户可以快速而顺畅地进行PC游戏。除了外观精美之外，惠普Firebird的体积也比较小，它是一款体积小功能强的计算机。</p>
<p><B>2010年：联想IdeaCentre A300</B></p>
<p>产品参数：2.2GHz Core 2 Duo处理器，500GB硬盘，分辨率为1920 x 1080的21.5英寸显示屏。</p>
<p>虽然苹果iMac几乎被视为一体机电脑的代名词，但是设计光滑的IdeaCentreA300证明，苹果并没有在一体机市场上一支独大。IdeaCentre A300的主机隐藏在显示屏下方的底座中，这使其看起来更加精致和时尚，它可以被摆放在家中的任何地方。</p>
<p><B>2011年：三星Chromebook Series 5</B></p>
<p>产品参数：1.66GHz Atom处理器，16GB固态硬盘，重量为3.3磅，分辨率为1280 x 800的12.1英寸显示屏。</p>
<p>与这里的其他PC机不同，三星Chromebook Series 5真正革命性的地方在于它的软件Chrome OS。谷歌和三星都押注，用户只需Web浏览器就能够完成日常的计算任务。Chromebook Series 5掀起了简装、廉价笔记本的浪潮，它不需要依赖于传统应用软件，就能够执行很多的任务。</p>
<p><B>2012年：苹果视网膜显示屏版MacBook Pro</B></p>
<p>产品参数：2.3GHz或2.6GHz Core i7处理器，256GB至768GB固态硬盘，重量为4.5磅，分辨率为2880 x 1800的15.4英寸显示屏。</p>
<p>苹果2012年版MacBook Pro只不过是在某些方面进行了小幅的改进，它仍然引领着笔记本电脑的设计潮流。2012年版MacBook Pro的亮点不仅仅在于锐利的视网膜显示屏，而且它也是首批摒弃光驱和硬盘驱动器的高端全尺寸笔记本电脑，它便于携带、运行快速。</p>
<p><B>2013年：宏碁Aspire R7</B></p>
<p>产品参数：1.8GHz Core i5处理器，500GB硬盘，分辨率为1920 x 1080的15.6英寸显示屏。</p>
<p>Windows 8触摸友好的用户界面掀起了变形本的到来，而宏基Aspire R7就是其中最为典型的代表，它是台式机、笔记本电脑或平板电脑的合体。不过，无论是作为台式机、笔记本电脑还是平板电脑，宏碁Aspire R7的表现都不是特别地好。但是，宏碁Aspire R7的到来表明，PC厂商非常迫切地希望阻止消费者购买移动平板电脑。</p>
<p><B>2014年：微软Surface Pro 3</B></p>
<p>产品参数：Core i3、i5或i7处理器，64GB至512GB固态硬盘，分辨率为2160 x 1440的12英寸显示屏。</p>
<p>如果你想要看看过去10年中PC机的变化，那么微软的最新旗舰级设备Surface Pro 3会让你一目了然。Surface Pro 3的键盘保护套模糊了平板电脑和笔记本之间的界限，它可以让用户抱着坐在沙发上看电影，也可以让用户在桌子上进行海量的媒体编辑。</p>
