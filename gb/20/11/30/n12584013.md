<tr><td VALIGN=TOP width="626"><h2 align=center>今年圣诞节“疫”乎寻常  南澳社交新规答疑</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2020/11/christmas-burnside-r-600x400.jpg" />
<h6>尽管面对防疫封锁和不断滚动更新的社交限制，但南澳仍在尽一切努力保持圣诞节活力，尽可能正常地欢度这个喜庆季节。（戴宁／大纪元）
</h6>
<hr>
<p>【大纪元2020年11月30日讯】（大纪元记者方君华澳洲阿德莱德编译报导）疫情时代，我们欢庆佳节的体验方式也不得不做些改变。</p>
<p>据《星期日邮报》（Sunday Mail）<ahref="https://github.com/bvbmfo3486/djy/blob/master/gb/tag/%E5%8D%97%E6%BE%B3.md#1">南澳</a>版报导，尽管面对防疫封锁和不断滚动更新的社交限制，人们正经历着巨大挑战，但南澳人仍在尽一切努力保持<ahref="https://github.com/bvbmfo3486/djy/blob/master/gb/tag/%E5%9C%A3%E8%AF%9E%E8%8A%82.md#1">圣诞节</a>活力，并确保所有人都能尽可能正常地欢度这个喜庆季节。</p>
<p><ahref="https://github.com/bvbmfo3486/djy/blob/master/gb/tag/%E5%8D%97%E6%BE%B3.md#1">南澳</a>州政府在12月1日（周二）放宽一系列的限制。从周一午夜起，南澳和维州之间的边界自3月关闭以来终获首次开放。</p>
<p>然而，这并不意味着我们的<ahref="https://github.com/bvbmfo3486/djy/blob/master/gb/tag/%E5%9C%A3%E8%AF%9E%E8%8A%82.md#1">圣诞节</a>会一切照常。</p>
<p>澳洲酒店协会（AHA）的霍恩（Ian Horne）表示，酒吧将以25％的接待能力运营，并仍需要兑现圣诞节前的预订，因此今年寻找场地可能超难。</p>
<p>不具备扫二维码功能的场馆，将需要执行更为严苛的限制，包括一人4平米的室内和室外的安全面积，以及100人的上限。</p>
<p>南澳商会（Business SA）首席执行官赫斯（Martin Haese）表示，当前的限制“可能对商家造成双重打击”。“在已经困难的一年之后，（限制接待能力）不仅会让商家失去重要的圣诞预订，企业也可能不得不取消圣诞节年尾活动，从而影响员工士气。”他说，“希望圣诞年会仍然可以举行，我们敦促政府每天审阅限制措施，如果健康情况持续好转，期待在12月14日之前放宽限制。”</p>
<p>下面以问答形式帮助您了解目前有关圣诞活动的各项<ahref="https://github.com/bvbmfo3486/djy/blob/master/gb/tag/%E9%98%B2%E7%96%AB%E9%99%90%E5%88%B6.md#1">防疫限制</a>。</p>
<ul>
<li><strong>圣诞节购物时有必要戴<ahref="https://github.com/bvbmfo3486/djy/blob/master/gb/tag/%E5%8F%A3%E7%BD%A9.md#1">口罩</a>吗？</strong></li>
</ul>
<p>从12月1日周二起，医务人员须强制戴<ahref="https://github.com/bvbmfo3486/djy/blob/master/gb/tag/%E5%8F%A3%E7%BD%A9.md#1">口罩</a>。其它情况下，戴口罩不是强制性的，但是如果您在公共场所（例如乘坐公共交通时）无法保持安全的1.5米<ahref="https://github.com/bvbmfo3486/djy/blob/master/gb/tag/%E7%A4%BE%E4%BA%A4%E8%B7%9D%E7%A6%BB.md#1">社交距离</a>的话，则应戴口罩。</p>
<ul>
<li><strong>在百货商店和</strong><strong>??</strong><strong>购物中心，还会有与圣诞老人合影活动吗？</strong></li>
</ul>
<p>有，但您必须先在线预订。孩子们将就座或站立，距离圣诞老人1.5米。每单预订最多可容纳六名家庭成员。</p>
<ul>
<li><strong>我的孩子可以坐在圣诞老人的膝盖上吗？</strong></li>
</ul>
<p>不可以，将采取严格的<ahref="https://github.com/bvbmfo3486/djy/blob/master/gb/tag/%E7%A4%BE%E4%BA%A4%E8%B7%9D%E7%A6%BB.md#1">社交距离</a>措施，因此与圣诞老人和他的“小精灵”之间不得有身体接触。</p>
<ul>
<li><strong>公司圣诞年会可以继续吗？</strong></li>
</ul>
<p>在许可的场所中最多可以容纳150人。虽然可以站立喝酒，但不允许跳舞，并且今年可能取消像卡拉OK和照相亭之类的活动，以保证安全的社交距离。所有员工在进入会场之前须确保使用州政府推出的二维码。</p>
<ul>
<li><strong>圣诞节聚会允许共享食物拼盘吗？</strong></li>
</ul>
<p>共享饮食没有限制。在餐桌上提供手部消毒剂将帮助降低风险。</p>
<ul>
<li><strong>在赌场玩游戏时，我可以和我的同伴在酒吧里喝啤酒或威士忌吗？</strong></li>
</ul>
<p>不可以。禁止在室内站立喝酒。</p>
<ul>
<li><strong>我可以在酒吧或餐厅吃圣诞节午餐吗？</strong></li>
</ul>
<p>可以。只要您遵守所有限制条件，例如最多只能预订10人，并且不能站立喝酒。但是业内人士表示，限制措施包括降低接待容量，因此今年找个场所可能比较困难，请务必提前预订。</p>
<ul>
<li><strong>如果我没有南澳政府的二维码、接触人跟踪应用程式或手机，是否会被拒绝进入许可的场所？</strong></li>
</ul>
<p>不会。如果顾客没有手机或应用程式，该场所将有纸笔记录作为备份。</p>
<ul>
<li><strong>我有一个大家庭，我们以往总是在家里分享圣诞午餐或晚餐。今年还能这样做吗？</strong></li>
</ul>
<p>在目前阶段，每户聚会限制为10人。不妨在圣诞节之前的几天中，安排几次不超过10人的圣诞节家庭聚会。</p>
<ul>
<li><strong>我可以按圣诞节传统拜访老年亲戚吗？</strong></li>
</ul>
<p>可以，只要同时拜访的人数少于10人。考虑戴上口罩作为预防措施。如果老人们在疗养院，请先与工作人员联系。</p>
<ul>
<li><strong>在圣诞节夜，我是否需要在饼干边放些手部消毒液？</strong></li>
</ul>
<p>这可能是个好办法来告诉孩子们：圣诞老人一边练习着防疫安全措施，一边继续进行着不可改变的规矩——圣诞节传统不会取消。</p>
<ul>
<li><strong>我可以在合唱团里唱颂歌吗？</strong></li>
</ul>
<p>看样子做不到，除非您可以戴着口罩唱歌。</p>
<ul>
<li><strong>我可以在圣诞节去教堂吗？</strong></li>
</ul>
<p>当前的限制意味着教堂服务仅限于150人。</p>
<ul>
<li><strong>我可以参观户外圣诞灯饰吗？</strong></li>
</ul>
<p>最好是开车行驶赏灯，建议远观。如不能与人保持安全社交距离，请戴上口罩。</p>
<ul>
<li><strong>如果我在圣诞节期间拜访维州的亲戚，是否需要在那里和</strong><strong>/或返回时隔离？</strong></li>
</ul>
<p>不需要。</p>
<p><strong>2020年12月1日起南澳放宽如下<ahref="https://github.com/bvbmfo3486/djy/blob/master/gb/tag/%E9%98%B2%E7%96%AB%E9%99%90%E5%88%B6.md#1">防疫限制</a>：</strong></p>
<p>已采用二维码系统的场所，取消顾客数量上限；</p>
<p>持酒牌场所室内每4平方米1人的规则仍保留；</p>
<p>持酒牌场所室外每2平方米1人的规则仍保留；</p>
<p>葬礼最多150人；</p>
<p>私人聚会最多150人；</p>
<p>婚礼最多150人（不变），但允许跳舞和站立饮酒；</p>
<p>在家举行私人聚会最多10人（不变）；</p>
<p>人数超过1000人的活动，必须有获得批准的防疫安全计划；</p>
<p>社区运动在设施程序修改后可恢复；</p>
<p>口罩，在无法保持社交安全距离的情况下需要。</p>
