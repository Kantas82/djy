<tr><td VALIGN=TOP width="626"><h2 align=center>公共卫生专家 : 边境管制需 “ 严格把关 ”</h2>

<h6></h6>
<hr>
<p>【大纪元2020年06月26日讯】( 大纪元记者章丰<ahref="https://github.com/cufnqn370/djy/blob/master/gb/tag/%E6%96%B0%E8%A5%BF%E5%85%B0.md#1">新西兰</a>编译报导 )据 RNZ 新闻媒体报导，一位<ahref="https://github.com/cufnqn370/djy/blob/master/gb/tag/%E5%85%AC%E5%85%B1%E5%8D%AB%E7%94%9F.md#1">公共卫生</a><ahref="https://github.com/cufnqn370/djy/blob/master/gb/tag/%E4%B8%93%E5%AE%B6.md#1">专家</a>支持反对党对中共病毒<ahref="https://github.com/cufnqn370/djy/blob/master/gb/tag/%E8%BE%B9%E5%A2%83.md#1">边境</a>管控的攻击，他说严格的<ahref="https://github.com/cufnqn370/djy/blob/master/gb/tag/%E6%A3%80%E7%96%AB.md#1">检疫</a>系统事关生死。</p>
<p>与此同时，政府也公布了一项强化隔离制度的计划，即对接待数千名从全球感染热点返回国民的一线员工进行例行病毒检测。政府称<ahref="https://github.com/cufnqn370/djy/blob/master/gb/tag/%E6%96%B0%E8%A5%BF%E5%85%B0.md#1">新西兰</a>的一些<ahref="https://github.com/cufnqn370/djy/blob/master/gb/tag/%E8%BE%B9%E5%A2%83.md#1">边境</a>管制条例是世界上最严格的，但这些管制措施现在还在进一步收紧。</p>
<p>卫生部长戴维 · 克拉克 ( David Clark ) 说，从现在起，所有机场工作人员、在有管理的隔离点或<ahref="https://github.com/cufnqn370/djy/blob/master/gb/tag/%E6%A3%80%E7%96%AB.md#1">检疫</a>酒店工作人员以及从机场接送旅客的司机，即使没有症状，也将以更高的比率接受中共病毒 ( COVID-19 ) 检测。</p>
<p>克拉克表示，根据新计划，从任何高风险国家返回的机组人员在获准回国前必须进行病毒检测。然而，除了美国，克拉克不能说出还有哪些国家可能被认为是感染病毒高风险地，因为形势变化太快了。</p>
<p>新西兰航空公司6月24日与员工会面，对新措施对业务的影响进行了讨论。这将是他们面临的新现实，因为边境限制可能会持续数月，甚至数年。</p>
<p>流行病学<ahref="https://github.com/cufnqn370/djy/blob/master/gb/tag/%E4%B8%93%E5%AE%B6.md#1">专家</a>迈克尔 · 贝克 ( Michael Baker ) 说，目前的措施让人放心，但他警告说，前面还有很长的路要走。 “ 我们现在有成千上万的人在许多不同的地点接受隔离，我们必须确保这些方案能够非常系统地应用于所有这些人群。”</p>
<p>距离大选只有几个月的时间，另一个政党可能会负责维护国家的防线。</p>
<p>国家党卫生发言人迈克尔 · 伍德豪斯 ( Michael Woodhouse ) 批评了最近几周的许多边境失误，并表示国家党执政的政府不会再犯同样的错误。</p>
<p>伍德豪斯说 : “ 在判断和处理过程上，这些绝对是令人震惊的失误，导致了中共病毒 ( COVID-19 ) 重返社会的风险大大增加。”</p>
<p>与此同时，贝克表示，反对党对政府应对中共病毒的持续批评是有道理的。“ 所有系统都运行良好是一个生死攸关的问题，所以我认为很多人都在努力研究这些措施以及它们被如何实施。”</p>
<p>他说，边境可能会出现更多的错误，因为隔离和检测机制将会持续很长时间。</p>
