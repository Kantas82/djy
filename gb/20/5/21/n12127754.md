<tr><td VALIGN=TOP width="626"><h2 align=center>疫情封锁下的新西兰对二氧化碳排放作出惊人影响</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2020/05/GettyImages-53546762-1-600x400.jpg" />
<h6>新西兰的统计数据 ( NIWA ) 证实，新西兰在肺炎疫情封锁下使二氧化碳水平大幅降低 41 %。图为 2001 年 5 月 20 日，在南坎特伯雷，南岛南部阿尔卑斯山日落的天空。( Ross Land / Getty Images )
</h6>
<hr>
<p>【大纪元2020年05月22日讯】( 大纪元记者章丰<ahref="https://github.com/ydbzqt347/djy/blob/master/gb/tag/%E6%96%B0%E8%A5%BF%E5%85%B0.md#1">新西兰</a>编译报导 )一项<ahref="https://github.com/ydbzqt347/djy/blob/master/gb/tag/%E7%A0%94%E7%A9%B6.md#1">研究</a>发现，新西兰的肺炎病毒<ahref="https://github.com/ydbzqt347/djy/blob/master/gb/tag/%E5%B0%81%E9%94%81.md#1">封锁</a>导致<ahref="https://github.com/ydbzqt347/djy/blob/master/gb/tag/%E4%BA%8C%E6%B0%A7%E5%8C%96%E7%A2%B3.md#1">二氧化碳</a>水平大幅<ahref="https://github.com/ydbzqt347/djy/blob/master/gb/tag/%E4%B8%8B%E9%99%8D.md#1">下降</a>了 41 %，这一降幅仅次于世界上一个国家。</p>
<p>这项<ahref="https://github.com/ydbzqt347/djy/blob/master/gb/tag/%E7%A0%94%E7%A9%B6.md#1">研究</a>于周四 ( <ahref="https://github.com/ydbzqt347/djy/blob/master/gb/tag/%E6%96%B0%E8%A5%BF%E5%85%B0.md#1">新西兰</a>时间 )发表在科学杂志《 自然气候变化 》( Nature Climate Change ) 上。该报告比较了 69 个国家、50 个美国州和 30 个中国省份的<ahref="https://github.com/ydbzqt347/djy/blob/master/gb/tag/%E4%BA%8C%E6%B0%A7%E5%8C%96%E7%A2%B3.md#1">二氧化碳</a>浓度，结果发现，除欧洲小国卢森堡外，新西兰的二氧化碳浓度降幅最大。</p>
<p>研究人员表示，这些国家和地区的人口占世界总人口的 85 %，二氧化碳总排放量的 97 %。研究发现，在极端的情况下，新西兰在<ahref="https://github.com/ydbzqt347/djy/blob/master/gb/tag/%E5%B0%81%E9%94%81.md#1">封锁</a>期间的二氧化碳排放量比去年同期<ahref="https://github.com/ydbzqt347/djy/blob/master/gb/tag/%E4%B8%8B%E9%99%8D.md#1">下降</a>了 41.1 %，而卢森堡下降了 44.6 %。另外哥伦比亚 ( 36.5 % )、立陶宛 ( 35.3 % )、尼日利亚 ( 34.2 % )和法国 ( 34 % ) 的排放量也大大低于世界平均降幅水平 26 %。而土库曼斯坦 ( 4.5 % )、哈萨克斯坦 ( 10.3 % )、乌克兰 ( 12.4 % ) 和爱沙尼亚 ( 12.5 % ) 的二氧化碳浓度变化是世界上最小的。</p>
<p>新西兰的统计数据 ( NIWA ) 证实了本周初期发布的研究报告，显示在封锁期间，奥克兰、惠灵顿和基督坊城的空气污染下降了 75 %。</p>
<p>空气质量科学家伊恩 · 朗利 ( Ian Longley ) 表示：“ 这一结果是前所未有的，将对公众健康产生重大影响。封锁的经验表明，只要我们停止开车，也就是在 4 级封锁刚开始的时候，空气质量立刻就会有变化。基本上，我们主要关注的是来自道路交通和其他来源的气体，如工业和航运，这已经下降了三分之二到四分之三，几乎所有被测量的地方都是如此。”</p>
<p>朗利表示，这项研究表明，通过集体努力可以做出重大的改变。</p>
