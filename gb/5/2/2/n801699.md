<tr><td VALIGN=TOP width="626"><h2 align=center>【投书】广州番禺区新造镇村民举报信</h2>

<h6></h6>
<hr>
	<p>【大纪元2月2日讯】举 报 人：广东省广州市<ahref="https://github.com/gkzlqr3061/djy/blob/master/gb/tag/%E7%95%AA%E7%A6%BA.md#1">番禺</a>区新造镇郭塱村、穗石村、北亭村、练溪村、贝岗村、南亭村村民</p>
<p>被举报人：广州市<ahref="https://github.com/gkzlqr3061/djy/blob/master/gb/tag/%E7%95%AA%E7%A6%BA.md#1">番禺</a>区新造镇广州大学城征地拆迁补偿安置工作办公室</p>
<p>案由：侵权纠纷</p>
<p>请求：成立专案组立即查处广州大学城违反国家法律“非法大圈地&#8221;、“城市化大跃进”、“害民强折”非法侵害农民权益的横行霸道行政行为。</p>
<p>广州大学城“非法大圈地”、“城市化大跃进”、“野蛮强拆民房”非法侵害农民权益的具体行政行为的开始于2002年1月9日，广州市规划局公告了《广州大学城发展规划》。按规划广州大学城将建在番禺区小谷围岛及南岸地区、规划范围约4 3.3平方公里，规划人口3 5至40万人(包括村镇人口)。</p>
<p>2 0 0 2年5月成立广州市番禺区新造镇广州大学城指挥部（现名为广州市番禺区新造镇广州大学城拆迁办公室）。广州市番禺区新造镇广州大学城指挥部于2 0 0 2年5月1 5日下达《广州大学城征地拆迁安置补偿办法》，该办法不但没有合法征地手续，而且补偿标准违反法律规定做低产值压低补偿。于是村民集体向广东省国土资源厅反映情况。但广东省国土资源厅没有答复村民该征地是否合法，而是委托番禺区国土资源和房屋管理局答复村民，该答复避重就轻，只字不谈征地是否合法及法律依据。而只是推销所谓拆迁方案。</p>
<p>2 0 0 2年6月广州市番禺区新造镇广州大学城指挥部在没有任何法定机关批准文件，也没有征地公告强行使用新造镇北约村l千多亩耕地(大部分是基本农田)用于建小谷围安置区（现名为谷围新村）。安置区从选址到施工都是唯一的地点，村民根本无权作选择。安置区用地违法问题村民已多次向省资源厅反映，但省国土资源厅一直对违法用地袒护、包庇。</p>
<p>广州大学城安置区建设用地未批先用，明显违反《土地管理法》第四十五条。广东省国土资源厅窥避责任，2 0 0 3年3月下发国土资(建)字[2 0 0 3]2 5号2 6号城镇建设用地批文，该批文也违反了《土地管理法》的征地程式和审批许可权。村民多次向省市有关部门反映情况，并要求广州市政府按国家的法律法规征地，保留行政村庄，不能毁灭有千年历史文化的村庄。对于小谷围村民来说，省市共建广州大学城，开发小谷围是福还是祸?</p>
<p>2 0 0 3年3月2 5日<B>广东省国土资源厅</B>突然下发对七条行政村和海珠，黄埔区在小谷围征地的征地公告。该建设用地为窥避审批许可权以不同的审批日期同时公告，将小谷围岛二万多亩耕地(其中一万五千多亩基本农田)化整为零分三十九个地块拆分审批。<br />2 0 03年3月6日广州市国土资源和房屋管理局下发《广州大学城征地拆迁安置补偿办法》下达后，举报人向广东省国土资源厅提出行政复议，但省国土厅拒不受理。广州市中级人民法院，和省高级人民法院也拒绝受理举报人的行政诉讼，又不作出不受理通知书。</p>
<p>举报人认为三十九个建设用地批文违反《中华人民共和国土地管理法》第四十五条，及《中华人民共和国城市规划法》第二十一条。大学城的工程项目，尤其涉及一万五千多亩的基本农田的征用，必须报国务院审批。所以举报人于2 0 0 3年4月依据《中华人民共和国行政复议法》第十二条向中国国土资源部提出行政复议，请求依法撤销广东省国土资源粤国土资(建)字[2 0 0 3]2 7号等3 5个建设用地批文，中国国土资源部拒绝受理并拒绝作出不予受理通知书和说明理由。</p>
<p>举报人依据《中华人民共和国行政复议法》规定，认为中国国土资源部拒不受理属于不作为的行为。于是举报人依据《中华人民共和国行政诉讼法》规定诉诸北京市中级人民法院。经法院责令后，中国国土资源部于2 0 0 3年5月2日依法受理举报人的行政复议。</p>
<p>依据《中华人民共和国行政复议法》复议机关应该2 0 0 3年7月2 3日作出决定书。查阅广东省国土资源厅提供给国土资源部的答辩和提供的证据，省国土资源厅并没有向国土资源部提供广东省人民政府征地文件，说明此征地行为是地方政府典型的“具体行政行为”。但中国国土资源部明知广州大学城建设征地没有合法批文，缺乏法律依据，中国国土资源部不但没有及时要求广东省国土资源厅停止执行其具体行政行为，而且在2 0 0 3年7月l 6日作出拖延至2 0 0 3年8月2 3日才作出复议决定的决定，旨在拖延时间。</p>
<p><B>而且更令人不解的是2 0 0 3年9月1日中国国土资源部下发终止复议通知书，但事实上举报人根本没有向国土资源部提出撤回复议。</B></p>
<p>举报人在行政复前和复议受理后都受到了打击报复、陷害、非法禁锢。地方执法人员耍特权，用人民专政的工具镇压手无寸铁的广大群众，政府动用警力迫村民就范、采用三强政策“强征、强迁、强拆”；还说征地拆迁绝对没有讨价还价的余地。政府动员村民迁出岛外定的是一个难以接受的补偿价：<B>房屋低于市场价五分之一，土地低于一河相隔海珠区相同地类六分之一，强行霸占土地。</B></p>
<p>广州市借撤销番禺市设立番禺区之机，通过调整区镇级土地利用规划，擅自改变基本农田的数量与布局，将小谷围的基本农田以所谓易地保护作托词，将大量的基本农田及耕地转为建设用地。</p>
<p>2003年l 0月2 7日广州市国有土地使用权出让预告广州大学城D S 1 5 O 7、S l 5 0 6、D S 1 5 1 0地块。用地性质为超市与办公楼、影剧院、酒店与会议中心，这三块土地7万多平方米属于小谷围岛的核心位置。也许是由于村民与传媒的压力，房管局至今一直未公开这几块地的开发人。</p>
<p>广州大学城总投资预计3 3 0亿元左右资金从哪里来?一部分是银行贷款，另一部分是省市拨款，学校根本无能力还款。地方政府热心于“办教育”，广州大学城“跨越式发展”最大的背后目的是“圈地”，地方政府一手征地，一手卖地，从村民中征用土地3至6万元一亩，有业内人士透露出让地块底价5 0 0万元左右一亩。这实际上是从农民手中掠夺财源为官员搞政绩工程。农民只有失去、失去还是失去！</p>
<p>2 0 0 3年五部委等部门组成督察组，对全国各省市自治区土地市场秩序治理督察，在一份((关于土地市场秩序治理整顿督察情况报告))中提到了广东省纠正广州市番禺区小谷围岛征用土地“化整为零&#8221;自行审批的做法撤消了4 1个相关批文。</p>
<p>2 0 0 4年1月2日《人民日报市场报》的十篇文章则直指广州大学城的“圈地问题”为“非法”。国土资源部执法监察局常嘉兴副局长在接受采访时告诉记者，广州市番禺区小谷围岛“化整为零”征用土地是广东省政府非法批地，所有审批文件无效，小谷围岛的征地要重新审批，到目前还没有收到广东省的报批材料。</p>
<p>但广州大学城一直没有停止施工，是什么人在支撑呢?答案只有一个是地方政府。<B>地方政府的权力是不是可以超越国务院的审批权呢? </B></p>
<p>广州大学城的开发地方政府控制媒体造势，还在有线电视放宣传片反复播放，“建设广州大学城合情合理合法”。</p>
<p>地方政府动用大量的警力恐吓，还动用警犬封锁施工现场，展开对小谷围万余农民声势浩大的开发建设征地及强拆强迁行动。村民开始意识大祸临头了。地方政府“非法圈地’’无所不克，拆迁大势所向披靡，浩浩荡荡，对于不配合拆迁的人采用株连式的打击报复，有的出外工作的村民被停工，开除，有的调离远处工作，还警告为了你前途不准回家乡。招工的不迁没有份。目的迫村民签定不平行的拆迁协定达到逼迁目的。村屋强拆前断电断电话，电视线路，严重威胁村民的生命财产，有的村民房屋门窗被打烂财物被窃，使村民处于极度的精神惶恐之中……地方政府的野蛮逼迁官员们得意洋洋还说“他们不搬迁，我们就这样看他们搬不搬”。</p>
<p>“无法无天”“横行霸道&#8221;“黑暗的野蛮拆迁”来临了。</p>
<p>2 0 0 3年1 2月1日到6日，广州市番禺区广州大学城拆迁安置办公室在没有人民政府裁决书，亦没有人民法院的强拆令的情况下，原副区长廖昌，新造镇书记张亮华，镇长杜锐钊，镇副书记梁健强，副镇长陈文华，镇农办主任潘立强，番禺区城管执法大队长、镇计生办主任刘少英等带领公安警察、消防武警，广州市城市管理执法大队番禺支队，辅警，治安员，出动粤警粤牌为4 0 6 8公安督察车，警察专用面包车，消防车，医院救护车，大客车等4 0多台，动用1千七百多警员，手持盾牌，长棒头戴头盔、拉着一群群气势汹汹的狼犬封锁进村的各个路口，把郭塱村重重围住。在房屋将要拆毁时村民们强烈要求出具合法执行文件；但官员们无法举证，说这是政府行为，是为了建大学城。村民们把自己反锁屋内而警察们气势汹汹地用大铁锤用力猛打破门而入，把我们的财物强行搬走。强行驱逐村民离开家园。十几台推土机象法西斯的坦克车一样把我们的房屋一间一间地夷为平地。处处粉尘盖天，各处重型机械的轰鸣声震耳欲聋，眼看着一间又一间的房屋倒下，我们的心象刀插一样一下比一下痛，眼在流泪喷火，心在流血、血在沸腾，我们当时2 0 0多人被分开赶到一边，只有抱着孩子无奈地看着痛着。“天啊……这是什么世界’，一个年近古稀的老妇人捶胸顿足地叫着哭着。有的人呆住了有的还晕倒过去被擡上救护车车走。</p>
<p>2 0 0 3年1 2月1日动用百多名警力野蛮强拆穗石赤坎村民房。</p>
<p>2 0 0 3年1 2月3日强拆的第三天，番禺区委书记徐金海乘坐在公安警用直升机上在郭望村上空盘旋观看清拆，地面上的官员用对讲机发出声说“向徐书记敬礼”顿时间警员们立正行军礼（一个市属区委书记俨然如国家元首，临空视察何等威风，地方诸侯潜越姿态顿现）；</p>
<p>2 0 0 3年1 2月6日动用几百名警察十台挖土机用同样的手法拆毁南亭村民房6 0多间；在短短的几天里，地方政府毁灭具有千年历史文化承传的村庄，美好的家园给地方政府滥用职权的长臂下化为废墟，空中只剩下我们无可奈何的声声悲叹。我们经历了切肉之痛的强拆，家当被抢走，一下子陷入了灾难之中。村民们失去了正常的工作和生活。但为了维护自己的合法权益，保护自己家园，维护自己的合法财产不被侵害，村民们决定要留在属于我们的土地上，搭起帐篷和棚舍继续生活继续走上访的路。</p>
<p>村民们走上了漫长的维权道路。依法向有关政府部门反映情况。<br />2 0 0 3年1 2月被野蛮强拆的郭塱、南亭、穗石、赤坎村民近五百人的难民中。自发组织3 2人到国家信访总局反映情况，我们受到了信访人员的接待，广东省驻北京办事处的陈勤耕和我们交谈，他说要尽快解决问题。</p>
<p>2004年1月8日晚上我们受到监视。<br />2004年1月1 1日晚上八时三十分，我们被地方政府人员警察强行押回广州火车站。越秀区公安分局数百多名警察将火车团团围住，并把我们强行押回番禺沙湾警察基地；将3 2人隔离逐一审问；2 8人扣押超过2 4小时；郭校伦，黄耀强被宣告拘留1 5天，理由是非法上访。在拘留期间每天饱受超过8小时的审问。精神折磨直到1月2 0日和2 1日先后释放郭校伦和黄耀强。</p>
<p>2 0 0 3年底到2 0 0 4年，地方政府在被毁郭塱村的宅基地上开挖人工湖，把残余的砖头瓦砾用泥土全部掩埋，在南亭村被毁民房上载花种草，将小谷围历代种植的成片成片百年荔枝林，龙眼林及改革开放后种上的果树毁于一旦。从此物产丰富，果实累累的渔米之乡、“绿色的小谷围&#8221;一去不复返了。</p>
<p>广州市政府毁大树种小树；<br />毁大果林造小杂树林；<br />毁了土地肥沃的基本农田搞宽阔的大马路；<br />把侵害农民利益说成公共利益；<br />现今的大学教育已经趋向产业化，地方政府把“公共利益”无限地扩大化“生拉硬套”。</p>
<p>大学城建设是地方官员的政绩，与房地产开发商的利益驱动联合形成变相土地开发。采用行政手段压低土地价格，再凭借行政压力要求大学埋单来树立“重视教育”的形象。</p>
<p>地方政府为了迫走村民用种种恶举，针对村民施行残害人性，摧毁人的尊严，将我们村民随意拘留、打伤、扣押、非法禁锢定罪、判刑。我们广大村民经历灾难的深重程度远在湖南嘉禾之上，逼迁者行为之恶劣也远在湖南嘉禾之上。</p>
<p>2004年2月8日拘留郭塱村民l 0人，拘留1 5天到1个月；有一人被判1 4个月刑期。事情是这样发生的：</p>
<p>2月6日，施工队将人工湖水抽到村民出入的村前大街上，污水灌进了村民的食水井。当时有二位村民想与旋工队论理，施工队保安员手持铁水管不由分说举起就打，另一位村民呼叫“施工队打人啊!”，村民被迫自卫因而发生冲突，冲突中，施工队保安及村民共有几人受轻伤，警方只听一面之词。当天，二十多名警员带着几位民工认人，到了2月8日下午四时二十分，天下着微丝细雨，寒风刺骨。我们村2 0 0多人正在漏水滴雨的棚舍内避寒，广州大学城拆迁办带着7 0 0多名身穿警用雨衣水鞋的警察，个个都手持盾牌警棍，将村民居住的六个地点分六组包围抽查，强行带走村民l 0人，村民质问警察为什么抓人，警察说：“涉嫌打伤施工人员。”本来抓人是警察的事，拆迁办的人来于什么?后来真相大白了，在番禺看守所审问村民时，拆迁办人员对村民说“只要你领上周转房的锁匙，我们什么也不追究了。”拆迁办为了达到目的采用非法手段，无非是想迫迁。大人被押上囚车，小孩们一边哭一边叫着“爸爸&#8221;，“妈妈”一边拉着父母的手。雨不停地下着，孩子们不停地哭着叫着，大人们痛骂着、所有人都是湿透了不知是雨还是泪。被毁灭破碎的村庄，破碎的家庭，破碎的心，2 0 0多村民顶着刺骨的寒风，走在破碎的村前大街上边走边流泪。“天啊!……为什么还要下雨!”</p>
<p>又一次野蛮强折又来了：<br />2 0 0 4年2月1 1日晚上我们收到消息得知拆迁办明天出动7 0 0多警察又来拆郭塱和南亭村的棚舍。经村民们的研究，为了减小损失决定全部转移，于是村民们由晚上1 2点～直忙到早上的5点才全部撤离。果然1 2号早上拆迁办官员们带领几百警员，辅警，用几台挖土机把我们的棚舍一扫而清。面对着无处不在的权力伤害，看见被毁的棚舍，木板，生活用品，家具的碎片，在被埋的土坑下半埋半露，心里不知是什么滋味。半个月后，我们又在另一个地方，也就是村的后面又重新搭起棚舍，坚持在这里居住。</p>
<p>2 0 0 4年3月被强拆户一行三十多人又第二次自发集体到北京国家信访总局上访。</p>
<p>2004年3月1 2日又被番禺公安带回番禺沙湾基地逐一隔离审问，其中上访人南亭的黄炳流在深夜审问时被“维护稳定”人员殴打，污辱，用白色纸巾把黄炳流的头部围了一圈，在额心点上红印接着就用拳打击胸部和腹部，黄炳流大叫救命，几个人才停了下来。</p>
<p>2 0 0 4年7月2日被强拆户一行4 0多人再次到北京上访。也就是第三次集体上京上访，同样受到了国家信访部门的接访。</p>
<p>2 0 0 4年7月1 4日村民回到广东省人民政府上访，省政府信访部门工作人员接访了我们五个代表。并将国土资源部交给我们转送广东省政府函交给接访人员，但接访人员拒绝接收，还叫我们拿到市政府去，还用很不耐烦的语气说“叫你们去还是不去”当我们请求出示广州大学城批文和申诉大学城是违法侵害时，接访的人员大声说：“我不与你们说法，说法的就到法院去，建设大学城要牺牲你们小谷围一万多人，要你们去死你们也要死！”，接访人员既不说自己姓名，又不配带工号，政府的形象成为了话柄。</p>
<p>地方政府违法征地一直动用大量警力镇压村民：<br />2003年4月1 5曰在穗石村；<br />2 0 0 3年5月9日在南亭村；<br />2 0 0 3年5月1 6日及1 8日在北亭村；<br />2 0 0 3年1 2月1至6日在郭塑村、南亭村；</p>
<p>最大几次出动警力，公安边警快艇，公安直升机，警车，囚车，警犬、电棒、高压水枪、盾牌警棍，镇压守法的村民，共出动人数三万多人次，中小规模次数难以统计。</p>
<p>大小事件拘留，拘捕几十人；<br />反复绑架三人；<br />电棒击昏人数十人，打伤十多人；<br />判刑三男一女；<br />践踏村民农作物近千亩，果树生态破坏最严重。</p>
<p>村民到省市政府上访多次，3—5人到北京上访达十次，被强拆户3 0—4 0人集体上京上访近三次。至今还没有解决问题，被强拆村民依旧住在棚舍里，迎来又一个寒冬，国务院三令五申要求要自查自纠。地方政府就广州大学城违法征地，害民拆迁的问题上根本没有纠正。他们还在传媒上大做文章，放纵村干部周游外国；排挤村民小组及村理财小组，使正常村务无法展开，使村民不知内情，继续顶风作案。。</p>
<p>横行霸道的野蛮强折又重来了：<br />2 004年8月1 2日那天气温很高，在防暴警，番禺区城管执法大队等几百人和挖土机的粗暴践踏下，郭塱村几经辛苦才重新搭建的棚舍又被夷平，并在地上挖满大坑使我们不能重搭帐棚，同样象上次一样把木板等材料埋在大坑里。在这次行动中警察们把上前理论的郭望村中年妇女<B>肖渐芬</B>抓起用力抛离原处摔落地，导致其脑震荡治疗费6 l 9 8元全部自己支付；与此同时一位十几岁的少女被打晕在地，被擡上救护车送院；还有位8 0多岁的老妇人被拆迁办官员骗走到了医院，说是带她看因抗迁入囚的儿子郭智华，但到了医院就没有人理了，又没有饭食，邻床的好心人见些情景才给一些面包点心给老人家吃。这样伤天害理的事，他们也敢做得出，难道社会主义新中国在太平盛世的社会，地方官员就可以为所欲为了吗?</p>
<p>后来在村民与镇政府交涉中，镇长杜锐钊还说：“我们没有打伤你们的人，以后再拆你们的棚可以随时拆，根本不用跟你们打招呼!”</p>
<p>从2 0 0 4年2月至8月为止，被强拆棚舍有5次之多。单是2 0 0 4年8月1 2日至1 3日两天内就连续强拆棚三次之多，虽然还下着暴雨，我们终于在8号夜晚十点四十分把最后一帐篷搭起来，那时候每个人都疲劳不勘，大人们抱起熟睡在木板上的孩子显得力不从心，女孩子的长发拖在地上湿透了，妇女们哭着那低低的哭声在静静的夜深中显得特别凄凉。</p>
<p>但村民凭着对我们热爱的中囯共产党的信任。凭着不屈不捞的民族精神，和邪不能胜正的信念及村民们的团结，至今拒绝与地方政府在征地拆迁补偿协定上签字。</p>
<p>广州市政府夸耀广州大学城建设掩盖了地方政府的违法违规行为，村民很清楚这是一个大骗局。据《羊城晚报》报道，说首期4 6亿元的征地拆迁款己全部进入番禺区财政局的账户。去年政府在传媒上又公告4 6亿元全部下发到补偿物件身上，我们知道1万4千多人的小谷围岛上村民六条行政村总征地拆迁费只不过十二亿元，这34亿元的差额哪里去了？</p>
<p>另一方面地方政府在媒体上说如何安置好村民，说什么“农民就业好”，其实从前的小谷围人丰衣足食，人人有菜种生活红火，只要肯干生活就无忧，但一下子变了，目前只有2千人安排在大学城干坐着无事做的“协管员”。其他人根本没有工作，生活没有来源，只能以后的事以后再算。现在是地方政府强取豪夺，严重损害农民权益被视为是正常的；而得到有效保障反而是奇怪的事。</p>
<p>地方政府为了骗村民说“在其他地方为各村买了1 5％的发展用地。”但直至现在，1 5％的发展用地不知道到哪里去了?据镇干部、村干部说，每一个村已付出数千万资金买发展用地，但己付了整年，地到现在不知去向，钱地就不翼而飞了。</p>
<p>2 0 0 5年1月发生了贝岗村干部挪用征地补偿资金4 0 0 0多万元人民币，用之委托证券公司炒作股票，这正是孳生腐败的体现。</p>
<p>地方政府顶风作案各自为政粗暴践踏宪法及法律，严重侵害国家利益，集体利益，人民利益。</p>
<p>特此强烈要求切实解决久拖未能解决的问题，归还被强拆户的宅基地，依法赔偿我们的损失。</p>
