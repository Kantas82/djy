<tr><td VALIGN=TOP width="626"><h2 align=center>新软件帮助管理储存电脑密码</h2>

<h6></h6>
<hr>
	<p>【大纪元3月28日讯】（大纪元记者秦飞编译报导）——美联社纽约报导，不可否认，你可能在所有的网站中使用同一个密码。你也应该知道，有人可能在一个不加密的网站中盗用了你的密码并在ｅＢａｙ上行骗，然后让你面对火冒三丈的购买人。</p>
<p>那么，现在你将如何处理你的密码？</p>
<p>今天的浏览器将帮助你储存密码，但仍未尽善尽美。如Ｉ．Ｅ．浏览器，任何一个接触你电脑的人都可以得到密码，包括那些安装隐藏间谍软件的黑客。</p>
<p>最近开放的Mozilla Firefox 浏览器在这方面有所提高，他们提供了一个初步的设想，把个人的密码和一个总密码锁定在一起。</p>
<p>另外一系列的密码管理程序也采用了同样的方法，只不过他们的更简便。这些程序能产生出一些随机的密码，这些密码因为无法被猜中而提高了安全性。</p>
<p>美联社记者测试了其中五项程序：Access Manager 2 (售价19.95美元)，TK8 safe (售价19.95美元), Password Agent 2 (售价24.95美元), Handy Password (售价30美元) 和 RoboForm Portable (售价39.98美元). </p>
<p>如果你忘记了总密码，至少你能在这些程序找到一个个人密码。</p>
<p>要在网上输入一个已储存的密码，你必须从密码管理程序中调出密码，再放入浏览器中。这比在浏览器中直接放入密码管理程序更为复杂。这些程序均宣称他们可通过键盘指令自动地输入密码，但在美联社记者的测试中并没有发现这项功能。</p>
<p>Access Manager 2，TK8 safe和Password Agent 2没有提供比Mozilla Firefox更先进的密码管理系统，但对那些为安全不介意花时间的Ｉ．Ｅ．用户来说，这些程序更为实用。</p>
<p>Handy Password更为先进一些：它在Ｉ．Ｅ．中建立了一条工具栏并自动地填进密码。也同时提供了简短方便的启动方法。而这条工具栏在其它浏览器中无法使用。</p>
<p>这些程序中的领先者是RoboForm Portable。与其它程序不同的是，它要求装入一个通用串行总线（ＵＳＢ）卡，这个微小的装置可以取代软盘片来装载文件。</p>
<p>这意味着你能随身携带密码和其它文件密码，RoboForm有１０ＭＢ的储存量。</p>
<p>装进了ＵＳＢ卡之后，你要打开文件并启动RoboForm程序，它把工具栏装置在Ｉ．Ｅ．，Firefox 或最近的Netscape 浏览器中。</p>
<p>当装进ＵＳＢ卡之后，如果程序能自动地启动将更为快捷，但没有这样运行。但当ＵＳＢ卡被移走之后，它会自动关机，使下一位接触电脑的人无法得到密码。</p>
<p>其它一些密码管理程序也有同样的安装ＵＳＢ卡的功能，但都没有RoboForm方便快捷。</p>
<p>请注意：如果你丢失了ＵＳＢ卡，你就丢失了你的密码（除非你把它拷贝到其它盘片或打印出来）。但是如果其他人得到了卡片，你的密码仍然是安全的，因为他们仍无法得到总密码。</p>
<p>同时，密码管理程序也无法保护电脑受网络诈骗，假网页等收集密码的影响。</p>
<p>RoboForm Portable值四十美元吗？如果你同时拥有几台电脑，答案是肯定的。如果你只有一台电脑，你可以选择Firefox或RoboForm　Ｐｒｏ，它们用硬盘代替ＵＳＢ卡，这可为你节省十美元。</p>
