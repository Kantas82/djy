<tr><td VALIGN=TOP width="626"><h2 align=center>追查国际指微软公司参与中共迫害人权</h2>

<h6></h6>
<hr>
	<p>【大纪元9月30日讯】(大纪元记者辛菲综合报导) 近日，包括雅虎(Yahoo)、Google和<ahref="https://github.com/zhafgp309/djy/blob/master/gb/tag/%E5%BE%AE%E8%BD%AF.md#1">微软</a>在内的网路公司，因配合中共进行网站内容的审查和监控而广受批评。</p>
<p>继9月7日记者无疆界组织披露，雅虎协助中共当局，将曾在电子邮件上谈论媒体管制的中国商报记者师涛定罪，“追查国际”组织今日(9月29日)发布了对<ahref="https://github.com/zhafgp309/djy/blob/master/gb/tag/%E5%BE%AE%E8%BD%AF.md#1">微软</a>协助中共当局迫害人权的最新调查报导。</p>
<p>早在2004年2月1日，英国报纸“The Observer”报道指出，有鉴于监控软体日渐被中共当局妄加使用，人权团体对微软公司对此问题所做的不当回应，提出强烈批评。</p>
<p>国际特赦组织研究员马克-艾礼森(Mark Allison)表示：“微软公司应更加关心人权被滥用的议题，并运用其高度的影响力提升表达对自由限制的关注。更重要的是，让那些遭迫害的人们早日迎向自由。但令人失望的是，他们似乎不甚重视这些人权问题”。</p>
<p>国际特赦组织曾表示，他们深信微软公司违反了针对多国商业往来的一项新的联合国人权条款，即“供应商应极力确保所出售的商品和服务，不能被用来伤害人权。”</p>
<p>“追查国际”组织针对微软公司参与中共迫害人权的事实调查报告内容概述如下：</p>
<p><b>美国高科技被中共利用来监控、拦截信息传送</b></p>
<p>2000年，一家自称在海外注册的中国软件公司──“海天软件”认为“Microsoft ISA Server”作为目前国际上领先的防火墙和代理服务器产品，在今天的宽带互联网市场上理应更有作为。</p>
<p>为了充分挖掘ISA Server的巨大潜力，微软(中国)有限公司选择其金牌解决方案提供商之一“海天软件有限公司”，开发完全基于Microsoft ISA Server 2000的企业上网安全管理套件。并且经过微软中国研发中心的联合测试，共同推出 Microsoft ISA Server 2000+ 海天上网管理套件整合计划。”</p>
<p>据微软(中国)网上的介绍，基于Microsoft的平台，开发的“海天网络邮件监控与管理系统(HiTeam Mail Monitor and Management System，简称HTMMS)， 被设计用来监控和管理各企业和事业单位的所有网络邮件通信，包括Outlook Web Access和主要的免费WebMail邮箱。”</p>
<p>其软件（HiTeam Mail Monitor，简称HTMM)的专业版“用在公安、安全、军队及重要的信息保密部门，在一台机器上可以同时监视1条或者多条1000Mbps的线路”，主要安装在城域网出口，“检查过往的邮件”和“自动截获邮件”。“HTMM专业版又叫做MailSieve，目前已经安装在多个大型城市的电信部门”。</p>
<p><b>微软、和光、海天三方共同推出网络监控新产品“ISA+EIM”</b></p>
<p>2003年8月初，FY04 微软分销商培训及渠道大会在厦门举行，和光微软产品线全体销售人员及各地的经销商参加了此次大会，其间，召开了“ISA+EIM”新品发布会及产品培训会，和光软件事业部总经理隋峥向与会者介绍了微软、海天、和光捆绑推广计划。拉开了和光与微软又一合作的序幕。</p>
<p>据和光集团介绍说，微软(中国)有限公司与海天软件有限公司合作推出基于Microsoft ISA Server 2000的企业员工上网行为管理系统，能够有效的对Web浏览，FTP下载，电子邮件收发等网络行为进行监控，即微软海天企业网络安全“双效”解决方案。和光是此“新包”的全国唯一代理。 </p>
<p><b>利用Microsoft ISA Server开发监控过滤软件控制学生上网</b></p>
<p>2002年1月7日，微软中国网发布Microsoft ISA Server 案例研究，介绍微软为中国的教育系统控制学生对Internet站点的访问选择Microsoft ISA Server开发过滤软件。</p>
<p>中共“陕西省教育工作委员会”给各高等学校下发了陕教工办[2003]49号文件，从中可了解中共教育系统对“法轮功”信息网上封堵的状况。</p>
<p>文件中说：中央和省委十分重视开展网上斗争，为了继续深入开展好与“法轮功”组织的斗争，继提出确保三项“零”指标之后，又把网上斗争情况作为衡量各地、各单位工作质量的重要指标。各校还要注意收集和及时上报网上斗争的有关情况，提供破案线索，并积极配合公安、安全部门追查访问“法轮功”网站。特别是下载、上传反宣品等有关材料的“法轮功”人员。</p>
<p>文件还提出了各校解决网上封堵信息的根本出路“要高度重视网络封堵技术的使用，加大资金和技术投入，及时采用高技术手段或进行技术升级和更新” 。</p>
<p><b>中共当局利用微软提供“安全”服务</b></p>
<p>1．公安部第三研究所是公安部门进行公共网络监管的技术支持单位，在公安部金盾工程的建设中负责针对国际互联网上各种信息进行监控的“金盾工程安全平台方案设计”。</p>
<p>2003年7月7日，公安部第三研究所与微软（中国）有限公司共同组建挂牌成立“公安部第三研究所──微软（中国）有限公司信息安全技术联合实验室”。</p>
<p>微软（中国）有限公司总裁唐骏表示，信息安全需要在政府的指导下，整个IT业界、研究机构和用户共同努力。我们希望通过此次与公安部第三研究所的合作，携手加强网络信息的安全，共同推动中国信息安全的发展。</p>
<p>为进一步获取更多的西方技术的支持，在同年10月16日，光明日报《信息化周刊》、中国信息安全产品测评认证中心、公安部第三研究所共同举办了“倡导国际合作促进国家信息安全”研讨会。</p>
<p>微软大中华区战略合作部总经理龚定宇表示：为什么要进行这两次合作？归根结底就是让我们的用户放心。把视窗源代码公布出来，让政府信赖的单位查看我们的源代码。我们希望通过这样的行动来消除大家的顾虑。 </p>
<p>从研讨会上还透露出一条信息：国际大公司要获得(中国)政府及涉密单位的订单，就必须向中方提供必要的技术资料，甚至要开放整个生产流程。</p>
<p>2．中共为达到全面性的信息封堵，建立覆盖全国范围的公共信息网络安全监控体系，推出“金盾工程”。</p>
<p>2001年 7月6日，“全面参与金盾工程信息化建设”的中国计算机软件和技术服务总公司（中软总公司）与微软开始在信息安全领域的合作，此举得到了包括微软首席执行官CEO史蒂夫鲍尔默等在内的微软高层的全力支持，更得到了国家计委、信息产业部、公安部和国家密码管理委员会等有关中共官方机构的大力支持。</p>
<p>中共公安部科技委主任李润森在“公安部科技委第四次全体会议”上讲话中提到“金盾工程”的建设内容主要包括信息网络建设、应用系统建设、标准规范建设、安全保障系统建设、运行管理体系建设、公共网络信息安全监控系统建设六大方面。“金盾工程”还有一项重要内容，就是建设 “全国信息网络安全监控中心” 。</p>
<p><b>“金盾工程”在网络迫害中所起的作用</b></p>
<p>2001年12月19日，贾春旺在北京召开的全国公安厅局长会议上作报告说 “全国公安机关在党中央、国务院和地方各级党委、政府的领导下，坚决贯彻落实中央的一系列重要指示精神，严厉打击、严密防范境内外敌对势力和“法轮功”X教组织的捣乱、破坏活动﹔要坚持科技强警，全面实施“金盾工程”，积极运用科学技术提高公安机关的战斗力”。</p>
<p>2002年6月23日《南方都市报》提到：“国家信息安全计算机测评中心主任屈延文教授透露：公安部的“金盾工程”中准备建立一个覆盖全国范围的公共信息网络安全监控体系，根据屈延文教授透露的这一方案，将对国际互联网上的各种信息监控，及时发现需要过滤的国外信息站点，对于国内的不良信息内容，及时制止传播”。</p>
<p>法轮功学员的电子邮件被中共当局认为严重威胁到了中共意识形态领域的主导权，一旦发现在国际互联网上下载文章、传递法轮功信息的便遭重判。</p>
<p>追查国际日前发布的调查报告指出，从1999年7月到2004年4月期间，据不完全统计，有108名法轮功学员由于上网而被监禁、劳教，三名法轮功学员被迫害致死。</p>
<p>中共公安部科技委主任李润森在“公安部科技委第四次全体会议”上声称金盾工程“在与‘法轮功’和其他敌对分子的斗争中，已经发挥了重要作用”。<font color=#ffffff>(http://www.dajiyuan.com)</font></p>
