<tr><td VALIGN=TOP width="626"><h2 align=center>美国最流行婴儿名字 各代人喜好不同</h2>

<h6></h6>
<hr>
	<p>【大纪元7月22日讯】 (大纪元记者吴翰林编译) 在公元1960年代，美国人为婴儿取的名字多半像是四月(April)和阳光(Sunshine)，与现代美国人命名大不相同。</p> <p>生活科学网站(LiveScience.com)报导，1950年代最热门的前十个名字和现在完全不一样，这里有一份美国社会安全局提供，在1950年代最流行男婴儿/女婴儿名字的名单。</p> <p>1. 詹姆斯/琳达 ( James / Linda)<br />2. 罗伯特/玛丽 (Robert / Mary)<br />3. 约翰/帕特丽 (John / Patricia)<br />4.麦可/芭芭拉(Michael / Barbara)<br />5. 大卫/苏珊 (David / Susan)<br />6. 威廉/南茜 (William / Nancy)<br />7.理查/黛博拉 ( Richard / Deborah)<br />8. 汤玛斯/珊卓拉(Thomas / Sandra)<br />9. 查理斯/卡罗 (Charles / Carol)<br />10. 盖理/凯思琳 (Gary / Kathleen)</p> <p>这里有去年最流行男婴儿/女婴儿名字的名单﹕</p> <p>1. 雅各/艾蜜莉 (Jacob / Emily)<br />2. 迈克 /伊莎贝拉 (Michael / Isabella)<br />3. 易申/艾玛 (Ethan / Emma)<br />4. 约书亚/埃娃 (Joshua / Ava)<br />5. 丹尼尔/麦迪森 (Daniel / Madison)<br />6. 克里斯多夫/苏菲亚(Christopher / Sophia)<br />7.安东尼/奥莉薇雅(Anthony / Olivia)<br />8. 威廉/艾碧嘉儿 (William / Abigail)<br />9.马太/汉娜 (Matthew / Hannah)<br />10. 安德鲁/伊莉莎白 (Andrew / Elizabeth)</p> <p>在公元1880年代到1950年代时，玛丽是最普遍的女婴的名字，而约翰在这一时期则是前五名男婴的名字。从公元1950年代一直到今日，迈克是父母们所喜爱的名字。</p> <p>自从1950年代起，美国婴儿名字的多样性有爆炸性的进展。在那个年代，四分之一的男孩和女孩的名字会是在那前十名最受欢迎的名字范围内。根据“婴儿姓名导览”的作者Laura Wattenberg表示，在最近几年，最受欢迎的前十个婴儿名字，只有占全部婴儿的姓名的十分之一。美国有越来越多样化的婴儿名字，因为现在有很多从日历或是古语来取名字。</p> <p>电影也可以让一个名字大受欢迎。从历史上来看，取名为麦迪逊的男孩比女孩还要多，到了公元1980年代，女孩取名麦迪逊这个名字的概率渐增。根据社会安全局资料，公元1990年代，麦迪逊这个名字成为女婴姓名排名的第29名。有人说，这是拜电影“美人鱼”(Splash)之赐，“美人鱼”是在1984年上映，美人鱼这个角色的名字就取名为麦迪逊 。***<br /><font color=#ffffff>(http://www.dajiyuan.com)</font></p>
