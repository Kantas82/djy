<tr><td VALIGN=TOP width="626"><h2 align=center>微软总部与惠普科技携手成立技术领航中心</h2>

<h6></h6>
<hr>
	<p>【大纪元4月10日讯】(2008年3月20日，<ahref="https://github.com/gmqafh324/djy/blob/master/gb/tag/%E5%BE%AE%E8%BD%AF.md#1">微软</a>台北) 看好台湾科技软实力及开创未来数位十年的产业竞争力，台湾微软今(二十)日宣布，与全球IT巨擘惠普科技共同在台成立“微软惠普联合技术领航中心”，此次合作是微软总部与惠普科技首次共同加码台湾，双方投注数千万先进软硬体设备并提供全球首席技术团队支援，以加速台湾产业运用灵活IT架构，为创造台湾下一波产业高峰做好万全准备，藉由协助台湾企业与产业合作伙伴在BI、 UC&#038;C、 Windows Server 2008解决方案上的技术升级，预估五年将带来相当于新台币25亿元的产值效益。</p> <p>“<ahref="https://github.com/gmqafh324/djy/blob/master/gb/tag/%E5%BE%AE%E8%BD%AF.md#1">微软</a>惠普联合技术领航中心” 引领台湾企业掌握未来数位十年营运优势</p> <p>新世代网路应用(Next Web)与动态灵活的IT架构(Dynamic IT)是台湾产业伙伴与企业掌握未来数位十年营运优势的关键。 微软技术中心全球总裁柯约翰(John Knutsen)表示，“微软总部及惠普科技看好台湾未来的科技‘软实力’，为了让台湾产业能进一步在新的科技趋势中脱颖而出，将透过‘微软惠普联合技术领航中心’全球的软、硬体技术资源与首席技术团队的专业咨询，提升台湾合作伙伴与企业用户开发下世代UC &#038;C整合通讯、BI商业智慧及Windows Server 2008解决方案的技术能力，进而为创造台湾下一波产业高峰做好万全准备!”</p> <p>惠普企业系统服务事业群策略联盟技术经理George Oakes表示，“惠普与微软的技术协助创造最佳营运成果，深为客户及合作伙伴仰赖。惠普的解决方案与概念验证将使客户与合作伙伴能有效评估各种IT环境中解决方案组合的效能与成果。”</p> <p>“一直以来，台湾微软非常重视产业伙伴及客户的声音，以期了解其对微软的期望与需求，为了协助产业伙伴与客户不断创新并创造全球竞争优势，微软每年编列了超过 65 亿美元投入研发。透过持续加码投资技术中心，台湾微软希望将更多研发成果与所有产业伙伴及客户分享。”微软技术中心总经理马大康指出：“微软技术中心目前已经成功的协助正航资讯、鼎新电脑、资通电脑、精诚资讯、博格科技、程曦资讯、伟盟系统、捷智商讯科技等30家独立软体开发商(ISV)将既有的应用解决方案导入Windows Server 2008。‘微软惠普联合技术领航中心’成立后，我们有信心可以协助更多的ISV及企业导入Windows Server 2008，预计协助60家ISV将既有的应用解决方案导入Windows Server 2008。”</p> <p>正航资讯技术总监林荣泰表示，最初的架构设计与最终的压力测试与概念验证是国内软体厂商软体开发解决方案遭遇的挑战，微软惠普联合技术领航中心在台成立后将完全解决这些问题，加速软体开发的时程。正航企业资源整合系统在微软技术团队的协助与微软技术中心先进的环境测试验证下，已经通过Works with Windows Server 2008 认证，并且能够完全在Windows Server 2008作业平台上高效能的执行，新的解决方案将为目前两岸三地超过60,000家的客户带来更具市场竞争力的优势。<br />延伸双雄实力，以双倍资源优化IT测试服务、降低开发风险</p> <p>“微软惠普联合技术领航中心”整合微软与惠普两大全球IT界巨擘的软硬体技术优势与全球资源，辅以全球首席技术团队的支援，不但能帮助台湾合作伙伴与企业用户使执行效能及安全性达到最佳化，同时也能缩减其学习新技术的时间与成本，加速解决方案开发的时程。</p> <p>‘微软惠普联合技术领航中心’拥有最先进的软硬体测试环境，包含Windows Server 2008、刀锋型服务器与高阶储存设备、多核心、虚拟化环境等等，能大大缩短解决方案测试的时间与成本。同时该中心亦将联合惠普的“技术解决方案中心”，延伸双方技术中心的资源，为企业提供一加一大于二的实力的优质服务，打造平稳高效能的核心基础IT架构。举例来说，一般解决方案中心无法展示异地备援或灾难复原(Disaster Recovery)的解决方案，透过双方解决方案中心的合作，便能营造分处两地的IT环境，清楚让客户了解评估建置此类解决方案的优势。此外，双方还能协助客户在开发软体上线前进行功能与压力测试，藉以优化应用软体的品质最佳化及压力测试以提供稳定的营运系统，达成软体开发风险的极致控管目标。</p> <p>聚焦业界主力需求，以软硬兼施的优势，成就企业最大价值</p> <p>如何整合多元复杂的通讯方式以及日以遽增的流通资讯，为企业获取最大价值的关键挑战。“微软惠普联合技术领航中心”亦提供客户结合网际网路与传统电信设备的整合通讯(Unified Communications，UC)解决方案，协助跨多重沟通机制多重区域的企业，能在任何时间、任何地点无障碍的完成沟通，几近完美的PBX整合，将能达到最佳营运绩效。另外，结合双方软硬体实力的商业智慧解决方案(BI)，将协助客户善用资讯，达到最佳的营运绩效、降低成本并缓和风险。客户可依IT环境复杂度的需求，从SQL Server 到Neoview平台，藉由测试以了解建置最合适的商业智慧解决方案。</p> <p>透过微软、惠普全球两大IT巨擘前所未有的紧密合作，相信将进一步协助台湾建立共荣产业结构，创造台湾资讯产业崭新里程碑!</p> <p>关于微软 </p> <p>微软公司成立于一九七五年，多年来在全球个人电脑与商用软体、服务与网际网路技术上居领导地位。MS-DOS、.NET、Office XP、 2007 Office system、Windows、Windows Server、Windows 3.0、Windows 95、Windows 98、Windows 2000、Windows XP、与Windows Vista都是微软公司的注册商标。此处所提到的其他产品与公司名称则可能是它们各别拥有者的商标。</p> <p>新闻联络人: <br />台湾微软公司 企业公关经理 黄盈绫<br
