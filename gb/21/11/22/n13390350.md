<tr><td VALIGN=TOP width="626"><h2 align=center>西澳新型金镍勘探公司诚邀投资者 联手发掘西澳矿藏新项目</h2>

<h6></h6>
<hr>
	<p>【大纪元2021年11月22日讯】（工商报导）Tambourah Metals Ltd(TMB)<span style="font-style: inherit; font-weight: inherit;">是一家总部位于西澳的金、镍和铂集团元素勘探公司</span><span style="font-style: inherit; font-weight: inherit;">——</span><span style="font-style: inherit; font-weight: inherit;">专注于评估、推进和开发西澳的矿产勘探项目，它于</span><span style="font-style: inherit; font-weight: inherit;">2021</span><span style="font-style: inherit; font-weight: inherit;">年</span><span style="font-style: inherit; font-weight: inherit;">8</span><span style="font-style: inherit; font-weight: inherit;">月</span><span style="font-style: inherit; font-weight: inherit;">12</span><span style="font-style: inherit; font-weight: inherit;">日注册成立。</span><span style="font-style: inherit; font-weight: inherit;">?</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">TMB</span><span style="font-style: inherit; font-weight: inherit;">勘探公司收购了西澳的两个黄金项目和两个镍项目，同时在澳大利亚证券交易所(</span><span style="font-style: inherit; font-weight: inherit;">ASX)</span><span style="font-style: inherit; font-weight: inherit;">上市并获得了</span><span style="font-style: inherit; font-weight: inherit;">800</span><span style="font-style: inherit; font-weight: inherit;">万澳元的融资，并且有能力使用该资金进行金矿勘探。拟议的资本结构是</span><span style="font-style: inherit; font-weight: inherit;">1300</span><span style="font-style: inherit; font-weight: inherit;">万澳元的市值。</span><span style="font-style: inherit; font-weight: inherit;">?</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">现在</span><span style="font-style: inherit; font-weight: inherit;">TMB</span><span style="font-style: inherit; font-weight: inherit;">诚招慧眼独具的投资者，在资源丰富的西澳矿区共同开拓勘探之路，挖掘仍待开发的矿藏财富。</span></p>
<figure id="attachment_13393019" aria-describedby="caption-attachment-13393019" style="width: 600px" class="wp-caption alignnone"><a target="_blank" href="https://i.epochtimes.com/assets/uploads/2021/11/id13393019-Tambourah-Metals-Project-Picture1.jpg"><img class="size-large wp-image-13393019" src="https://i.epochtimes.com/assets/uploads/2021/11/id13393019-Tambourah-Metals-Project-Picture1-600x856.jpg" alt="" width="600" b="856" /></a><figcaption id="caption-attachment-13393019" class="wp-caption-text"><ahref="https://github.com/bejthd314/djy/blob/master/gb/tag/%E8%A5%BF%E6%BE%B3%E5%8B%98%E6%8E%A2.md#1">西澳勘探</a>公司Tambourah Metals Ltd收购了西澳的两个黄金项目和两个镍项目。（商家提供）</figcaption></figure>
<p><span style="font-style: inherit; font-weight: inherit;">?</span></p>
<h2><span style="font-style: inherit; font-weight: inherit;">TMB</span><span style="font-style: inherit; font-weight: inherit;">项目介绍</span><span style="font-style: inherit; font-weight: inherit;">?</span></h2>
<p><span style="font-style: inherit; font-weight: inherit;">1.</span><span style="font-style: inherit; font-weight: inherit;">黄金项目：</span><span style="font-style: inherit; font-weight: inherit;">?</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">Tambourah</span><span style="font-style: inherit; font-weight: inherit;">金矿项目位于西澳的皮尔巴拉（</span><span style="font-style: inherit; font-weight: inherit;">Pilbara</span><span style="font-style: inherit; font-weight: inherit;">）东部地区的</span><span style="font-style: inherit; font-weight: inherit;">Marble Bar</span><span style="font-style: inherit; font-weight: inherit;">。这里是历史悠久的</span><span style="font-style: inherit; font-weight: inherit;">Tambourah</span><span style="font-style: inherit; font-weight: inherit;">采金区域，位处</span><span style="font-style: inherit; font-weight: inherit;">Port Hedland</span><span style="font-style: inherit; font-weight: inherit;">和</span><span style="font-style: inherit; font-weight: inherit;">Newman</span><span style="font-style: inherit; font-weight: inherit;">之间。该项目已获得一份勘探许可证(</span><span style="font-style: inherit; font-weight: inherit;">Exploration Licence)</span><span style="font-style: inherit; font-weight: inherit;">和四份探矿许可证(</span><span style="font-style: inherit; font-weight: inherit;">Prospecting ?Licence)</span><span style="font-style: inherit; font-weight: inherit;">。</span><span style="font-style: inherit; font-weight: inherit;">这些矿权总面积为</span><span style="font-style: inherit; font-weight: inherit;">696</span><span style="font-style: inherit; font-weight: inherit;">公顷，具有极高的潜在金矿化前景。</span><span style="font-style: inherit; font-weight: inherit;">?</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">TMB</span><span style="font-style: inherit; font-weight: inherit;">另一个颇具开发前景的项目是</span><span style="font-style: inherit; font-weight: inherit;">Cheela</span><span style="font-style: inherit; font-weight: inherit;">金矿项目。该项目位于西澳南部皮尔巴拉地区，它已经获得两个勘探许可证，矿权面积占地</span><span style="font-style: inherit; font-weight: inherit;">38,158</span><span style="font-style: inherit; font-weight: inherit;">公顷。</span><span style="font-style: inherit; font-weight: inherit;">?</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">2.</span><span style="font-style: inherit; font-weight: inherit;">镍项目：</span><span style="font-style: inherit; font-weight: inherit;">?</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">TMB</span><span style="font-style: inherit; font-weight: inherit;">在</span><span style="font-style: inherit; font-weight: inherit;">Julimar Nth-Jimperding</span><span style="font-style: inherit; font-weight: inherit;">拥有大量土地。</span><span style="font-style: inherit; font-weight: inherit;">Julimar North</span><span style="font-style: inherit; font-weight: inherit;">项目位于西澳小麦带(</span><span style="font-style: inherit; font-weight: inherit;">Wheatbelt</span><span style="font-style: inherit; font-weight: inherit;">）地区。</span><span style="font-style: inherit; font-weight: inherit;">Julimar</span><span style="font-style: inherit; font-weight: inherit;">项目也获得两个勘探许可证及正在申请两个勘探许可证，矿权面积为</span><span style="font-style: inherit; font-weight: inherit;">50,828</span><span style="font-style: inherit; font-weight: inherit;">公顷(</span><span style="font-style: inherit; font-weight: inherit;">Tambourah ?Metals ?80%)</span><span style="font-style: inherit; font-weight: inherit;">。</span><span style="font-style: inherit; font-weight: inherit;">?</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">公司最近在附近发现了</span><span style="font-style: inherit; font-weight: inherit;">Julimar PGE-Ni-Cu-Co</span><span style="font-style: inherit; font-weight: inherit;">矿床。铂族元素(</span><span style="font-style: inherit; font-weight: inherit;">PGE)</span><span style="font-style: inherit; font-weight: inherit;">中的铂(</span><span style="font-style: inherit; font-weight: inherit;">Pt)</span><span style="font-style: inherit; font-weight: inherit;">和钯（</span><span style="font-style: inherit; font-weight: inherit;">Pd</span><span style="font-style: inherit; font-weight: inherit;">）可用于汽车的净化转化器和锂电池中。</span><span style="font-style: inherit; font-weight: inherit;">?</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">TMB</span><span style="font-style: inherit; font-weight: inherit;">在</span><span style="font-style: inherit; font-weight: inherit;">Achilles</span><span style="font-style: inherit; font-weight: inherit;">位于</span><span style="font-style: inherit; font-weight: inherit;">Gerry Well</span><span style="font-style: inherit; font-weight: inherit;">绿岩带。该绿岩带以南位于</span><span style="font-style: inherit; font-weight: inherit;">Duketon</span><span style="font-style: inherit; font-weight: inherit;">绿岩带。</span><span style="font-style: inherit; font-weight: inherit;">TMB</span><span style="font-style: inherit; font-weight: inherit;">公司最近的一项研究发现，在南部毗邻的</span><span style="font-style: inherit; font-weight: inherit;">Collurabbie</span><span style="font-style: inherit; font-weight: inherit;">项目区域，可能存在类似的</span><span style="font-style: inherit; font-weight: inherit;">PGE-Ni-Cu</span><span style="font-style: inherit; font-weight: inherit;">的矿化表现。</span></p>
<h2><span style="font-style: inherit; font-weight: inherit;">公司执行董事</span><span style="font-style: inherit; font-weight: inherit;">Rita Brooks</span><span style="font-style: inherit; font-weight: inherit;">的淘金之路</span></h2>
<p><span style="font-style: inherit; font-weight: inherit;">80</span><span style="font-style: inherit; font-weight: inherit;">年代早期是黄金蓬勃发展的时期，因此吸引了不少澳人加入淘金潮，</span><span style="font-style: inherit; font-weight: inherit;">?TMB</span><span style="font-style: inherit; font-weight: inherit;">公司执行董事</span><span style="font-style: inherit; font-weight: inherit;">Rita Brooks</span><span style="font-style: inherit; font-weight: inherit;">与丈夫</span><span style="font-style: inherit; font-weight: inherit;">Bill</span><span style="font-style: inherit; font-weight: inherit;">就是早期加入淘金潮的勘探者之一。</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">三十年前，</span><span style="font-style: inherit; font-weight: inherit;">Brooks</span><span style="font-style: inherit; font-weight: inherit;">辞去了她在墨尔本美术老师的工作，与丈夫只身前往西澳的金矿地区</span><span style="font-style: inherit; font-weight: inherit;">——Goldfield</span><span style="font-style: inherit; font-weight: inherit;">开始了黄金勘探生涯。</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">作为一名业余勘探员，</span><span style="font-style: inherit; font-weight: inherit;">Brooks</span><span style="font-style: inherit; font-weight: inherit;">如何做到慧眼识宝藏呢？</span><span style="font-style: inherit; font-weight: inherit;">?</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">Brooks</span><span style="font-style: inherit; font-weight: inherit;">表示，首要的工作就是搜集信息，我会把不同时期发现的矿藏的信息结合起来，以形成新的地质图用来勘探。</span><span style="font-style: inherit; font-weight: inherit;">?</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">100</span><span style="font-style: inherit; font-weight: inherit;">年前的地质报告是手绘的，并非数据编制，所以许多信息并没有保留下来。因此，</span><span style="font-style: inherit; font-weight: inherit;">Brooks</span><span style="font-style: inherit; font-weight: inherit;">还是喜欢使用古老的地质图进行勘探研究以寻找被遗忘的宝藏。</span><span style="font-style: inherit; font-weight: inherit;">?</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">在</span><span style="font-style: inherit; font-weight: inherit;">2000</span><span style="font-style: inherit; font-weight: inherit;">年的中期，</span><span style="font-style: inherit; font-weight: inherit;">Brooks</span><span style="font-style: inherit; font-weight: inherit;">上市了几家公司，其中包括</span><span style="font-style: inherit; font-weight: inherit;">Berkeley Resources</span><span style="font-style: inherit; font-weight: inherit;">，该公司是现在的一家铀矿（</span><span style="font-style: inherit; font-weight: inherit;">Uranium</span><span style="font-style: inherit; font-weight: inherit;">）公司</span><span style="font-style: inherit; font-weight: inherit;">—Berkeley Energia</span><span style="font-style: inherit; font-weight: inherit;">。</span><span style="font-style: inherit; font-weight: inherit;">?</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">在过去</span><span style="font-style: inherit; font-weight: inherit;">10</span><span style="font-style: inherit; font-weight: inherit;">年的大部分时间里，澳洲的上市勘探企业在筹集勘探资金方面进展缓慢。但</span><span style="font-style: inherit; font-weight: inherit;">Brooks</span><span style="font-style: inherit; font-weight: inherit;">依然持续在西澳积累项目，其中四个项目被出售给新公司</span><span style="font-style: inherit; font-weight: inherit;">Tambourah Metals Ltd Tambour (TMB)</span><span style="font-style: inherit; font-weight: inherit;">。</span><span style="font-style: inherit; font-weight: inherit;">?</span></p>
<p><span style="font-style: inherit; font-weight: inherit;">Brooks</span><span style="font-style: inherit; font-weight: inherit;">说，</span><span style="font-style: inherit; font-weight: inherit;">Tambourah</span><span style="font-style: inherit; font-weight: inherit;">已经吸引了投资者的强烈兴趣。“鉴于那些成功的投资案例，这个（黄金勘探）市场正在火热起来。我们迫不及待地要继续投入实地勘探了。”</span><span style="font-style: inherit; font-weight: inherit;">?</span></p>
