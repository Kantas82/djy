<tr><td VALIGN=TOP width="626"><h2 align=center>2021年阿市五大投资潜力区 阿德莱德港居首</h2>

<h6></h6>
<hr>
<p>【大纪元2021年03月03日讯】（大纪元记者谢茹慧澳洲<ahref="https://github.com/qqypmv3232/djy/blob/master/gb/tag/%E9%98%BF%E5%BE%B7%E8%8E%B1%E5%BE%B7.md#1">阿德莱德</a>编译报导）对于有意在2021年<ahref="https://github.com/qqypmv3232/djy/blob/master/gb/tag/%E6%8A%95%E8%B5%84.md#1">投资</a><ahref="https://github.com/qqypmv3232/djy/blob/master/gb/tag/%E6%88%BF%E5%B8%82.md#1">房市</a>的买家来说，<ahref="https://github.com/qqypmv3232/djy/blob/master/gb/tag/%E9%98%BF%E5%BE%B7%E8%8E%B1%E5%BE%B7.md#1">阿德莱德</a>是一个颇有<ahref="https://github.com/qqypmv3232/djy/blob/master/gb/tag/%E6%BD%9C%E5%8A%9B.md#1">潜力</a>的市场。<ahref="https://github.com/qqypmv3232/djy/blob/master/gb/tag/%E6%88%BF%E5%B8%82.md#1">房市</a>分析专家，Hotspotting网站的创立人赖德（Terry Ryder）认为阿德莱德是全澳最被低估的市场，其中有五个市政辖区的房价可望在今年实现最强劲增长。</p>
<p>来自阿德莱德时事信息网的报导，赖德在一份最新的报告中列出了阿德莱德2021年五大<ahref="https://github.com/qqypmv3232/djy/blob/master/gb/tag/%E6%8A%95%E8%B5%84.md#1">投资</a>热区，排在首位的是阿德莱德港恩菲尔德（Port Adelaide Enfield）市政区，接下来依次是Playford、Mitcham、Tea Tree Gully和Salisbury。<ahref="https://github.com/qqypmv3232/djy/blob/master/gb/tag/%E6%BD%9C%E5%8A%9B.md#1">潜力</a>评估因素包括房价<ahref="https://github.com/qqypmv3232/djy/blob/master/gb/tag/%E5%8F%AF%E8%B4%9F%E6%8B%85%E6%80%A7.md#1">可负担性</a>、购买和租赁需求、配套设施便利性以及新的就业中心。</p>
<p>阿德莱德港恩菲尔德区是赖德最看好的投资潜力区。除了平易的房价及正在攀升的需求之外，正在进行的航空母舰项目以及南澳政府的一项大型就业迁移计划是推高房价的主要动力。</p>
<p>赖德在报告中说，这两个大型项目，加上若干小型项目将为当地带来新的就业机会，加速该区从传统工业区转型为新兴的住宅、商业和旅游中心。</p>
<p>市内的中位房价因具体郊区有所不同，Rosewater的中位价目前仅30万，一些滨海区的房价达到60万，从总体上仍相对平易。赖德说，无论在哪个区投资置业，现在都能获得很好的性价比。</p>
<p>Ray White房产连锁公司阿德莱德港分店的中介萨罗斯（Nick Psarros）表示，阿德莱德港及其周边郊区在过去几年改观显着，吸引了越来越多的居民。他说，这里曾经以工业为主，很多物业年久失修，但现在重振活力。他说：“新活力的注入也吸引了很多买家迁入，寻找投资机会。”</p>
<p>他还表示，买家看中的是便宜的房价以及旧物业的翻新机会，同时也正是这些翻新的酒吧和咖啡馆、新开发的住宅物业大大改善了区内的宜居性。</p>
<h1>五大投资热区的潜力分析</h1>
<h2>Port Adelaide Enfield</h2>
<ul>
<li>国防航空母舰建造合同以及南澳政府迁移计划将大大推动区内的经济和就业机会；</li>
<li>过去一年房价和需求大幅上升；</li>
<li>中位价从30万至60万不等；</li>
<li>区内空置率不到1%，租金收益率在3.5-5%。</li>
</ul>
<h2>Playford</h2>
<ul>
<li>北联高速（Northern Connector）的开通大大便利了北阿德莱德和主要货运交通设施的接驳，提升了当地的商贸机会；</li>
<li>皇家空军基地和Lyell McEwin医院为当地最大企业；</li>
<li>区内大部分郊区的中位房价在20万至30万之间，正处于上升通道；</li>
<li>租金回报率高，部分区达到6.5%，大部分区的空置率不到1%。</li>
</ul>
<h2>Mitcham</h2>
<ul>
<li>与阿德莱德市中心相距15公里，属于中环区，中位房价在50万至70万；</li>
<li>空置率属全澳最低，部分邮编区为零空置率，房价也相应处于上升压力；</li>
<li>毗邻若干主要就业中心，包括阿德莱德市中心、弗林德斯（Flinders）医疗和教育区以及汤斯利（Tonsley）创新中心，区内失业率也相对较低；</li>
<li>正在扩建的健康和教育项目——投资总额达15亿的弗林德斯村（Flinders Village）——将大大提升当地经济和配套。</li>
</ul>
<h2>Tea Tree Gully</h2>
<ul>
<li>区内中位房价在35万至55万之间，大多数郊区房价在去年稳定上涨；</li>
<li>租赁收益率在3-6%之间，空置率不到1%；</li>
<li>与阿德莱德市中心，以及其它就业中心，包括阿德莱德港、毗邻的爱丁堡（Edinburgh）和莫森湖（Mawson Lakes）接驳便利；</li>
<li>区内失业率低，购房需求强劲。</li>
</ul>
<h2>Salisbury</h2>
<ul>
<li>Salisbury市政区总体正迎来开发营建高潮，大大推动了当地经济，创造了数万就业机会；</li>
<li>毗邻若干关键就业中心，包括食品饮料加工区、国防、网络安全和太空，以及一些新兴行业也吸引了大量企业，创造了不少就业机会；</li>
<li>中位房价在30万至45万左右，由于需求不断增长，房价也可望在今年上涨；</li>
<li>租金回报率在4-6%之间，绝大多数区空置率不到1%。</li>
</ul>
