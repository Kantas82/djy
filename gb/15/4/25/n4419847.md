<tr><td VALIGN=TOP width="626"><h2 align=center>乔州大学秋季开始涨学费</h2>

<h6></h6>
<hr>
	<p>【大纪元2015年04月25日讯】【大纪元记者王竹亚特兰大综合报导】根据乔治亚大学系统董事会星期二批准的改动方案，今年秋季开始，乔州的30所公立大学学费都将上涨。</p>
<p>各学校学费增幅不同，其中20所大学系统的涨幅为2.5％，其余10所学校的学费增幅为3％到9％。乔州理工学院和乔州大学这样的科研型机构，学费涨幅最高，亚特兰大大都会学院和乔州中部州立学院上个月被批准为研究型大学。</p>
<p>根据不同的学校，2.5％的增幅对于州内本科学生相当于每学期增加33到64美元，而外州学生则增加126至225美元。四个研究型机构的大学涨幅从221到405美元不等，乔州理工学院涨幅最高。</p>
<p>学校系统的官员表示，学费上涨有诸多原因，主要是涉及到所需要的资源，希望能减少每班学生人数、留住和招聘高质量的教师以及维持学校日常的运营费用等。</p>
<p>乔州30所大学学费增加如下：</p>
<p>乔治亚理工学院(Georgia Tech)、乔治亚大学(University of Georgia)、亚特兰大大都会州立学院(Atlanta Metropolitan State College)、乔州中部州立大学(Middle Georgia State University)：9％</p>
<p>乔州谷内学院(Georgia Gwinnett College)：8.3％<br />乔治亚摄政大学(Georgia Regents University)、乔治亚州立大学( Georgia State University)：5.5％<br />北乔治亚大学(University of North Georgia)：5％<br />肯尼索州立大学(Kennesaw State University)：4.4％<br />乔治亚学院与州立大学(Georgia College &#038; State University)：3％</p>
<p>以下学校学费均增涨2.5％：<br />南乔治亚大学(Georgia Southern University)<br />奥尔巴尼州立大学(Albany State University)<br />瓦尔多斯塔州立大学(Valdosta State University)<br />阿姆斯特朗大西洋州立大学(Armstrong Atlantic State University)<br />克莱顿州立大学(Clayton State University)<br />哥伦布州立大学(Columbus State University)<br />谷堡州立大学(Ft. Valley State University)<br />乔州西南州立大学(Georgia Southwestern State University)<br />萨凡纳州立大学(Savannah State University)<br />西乔治亚大学(University of West Georgia)<br />亚伯拉罕鲍德温农学院(Abraham Baldwin Agricultural College)<br />乔治亚海滨学院(College of Coastal Georgia)<br />多尔顿州立学院(Dalton State College)<br />戈登州立学院(Gordon State College)<br />班布里奇州立学院(Bainbridge State College)<br />达顿州立学院(Darton State College)<br />东乔治亚州立学院(East Georgia State College)<br />乔州海兰德学院(Georgia Highlands College)<br />乔州派瑞米特学院(Georgia Perimeter College)<br />南乔治亚州立大学(South Georgia State College)</p>
<p>（数据来源：乔治亚大学系统）</p>
