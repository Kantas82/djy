<tr><td VALIGN=TOP width="626"><h2 align=center>7.5级地震致纽西兰各地奔忙</h2>

<h6></h6>
<hr>
	<p>【大纪元2016年11月14日讯】（大纪元记者李心然<ahref="https://github.com/yqruso3957/djy/blob/master/gb/tag/%E7%BA%BD%E8%A5%BF%E5%85%B0.md#1">纽西兰</a>编译报导）今日凌晨纽西兰南岛发生7.5级<ahref="https://github.com/yqruso3957/djy/blob/master/gb/tag/%E5%9C%B0%E9%9C%87.md#1">地震</a>后，各地的受损状况及工作进展情况目前如下：</p>
<ul>
<li>Kaikura和几个南岛小镇目前无法与外界联系，供水和污水处理系统中断，道路与铁路被阻。</li>
</ul>
<ul>
<li><ahref="https://github.com/yqruso3957/djy/blob/master/gb/tag/%E5%9C%B0%E9%9C%87.md#1">地震</a>导致至少两人死亡。一名受害者死于心脏病突发。另一名受害者在Kaikura因房子倒塌致死。</li>
</ul>
<ul>
<li>首都惠灵顿受到重创，建筑物和道路遭到损坏，民众被告知不要去中心商务区。</li>
</ul>
<ul>
<li>南岛和北岛南部在12小时内发生250多次余震。</li>
</ul>
<ul>
<li>居民在海啸预警后逃离家园，有部分空置的房屋被窃贼洗劫，警方目前在调查发生的三宗案件。</li>
</ul>
<ul>
<li>高中统考（NCEA）面临中断和延迟，奖学金考试（scholarship exam）今天暂停。</li>
</ul>
<ul>
<li>一处大型的山体滑坡导致到Waiau方向的70号高速公路被阻断，目前无法通行。</li>
</ul>
<ul>
<li>111紧急电话服务在最重灾区因通路问题有延迟。</li>
</ul>
<ul>
<li>地震造成的损失预计高达上亿元。</li>
</ul>
<ul>
<li>坎特伯雷北部和惠灵顿的学校在损失评估完成前被告知停课。</li>
</ul>
<ul>
<li><ahref="https://github.com/yqruso3957/djy/blob/master/gb/tag/%E7%BA%BD%E8%A5%BF%E5%85%B0.md#1">纽西兰</a>整个东海岸已经取消海啸警告，但住在海岸边的人被警告要远离沙滩和海岸线。</li>
</ul>
<ul>
<li>消防已派出7人一组的搜救队从基督城到Kaikura进行搜救。</li>
</ul>
<ul>
<li>总理约翰·凯伊（John Key）已取消了对阿根廷进行正式访问，以应对地震余波。</li>
</ul>
<ul>
<li>总理表示，目前通信困难，对损害及伤亡的全面评估正在进行中。</li>
</ul>
<ul>
<li>多处地区断电。</li>
</ul>
<ul>
<li>纽西兰运输局（NZTA）表示，会关闭一些主要公路和道路。</li>
</ul>
