<tr><td VALIGN=TOP width="626"><h2 align=center>瑞典出新法 允警方监听 禁资助恐怖组织</h2>

<h6></h6>
<hr>
	<p>【大纪元2016年04月12日讯】（大纪元记者李尚综合报导）瑞典有关禁止出国旅行参与恐怖犯罪活动的法律出台。如有瑞典人出国旅行的目的是实施或准备与恐怖犯罪相关的活动，提供或接受恐怖犯罪培训都属于犯法，将被判刑入狱。</p>
<p><span style="font-family: 'Arial Unicode MS';">周五</span>(4<span style="font-family: 'Arial Unicode MS';">月</span>1<span style="font-family: 'Arial Unicode MS';">日</span>)<span style="font-family: 'Arial Unicode MS';">，有关瑞典人出国旅行参与恐怖活动的法律禁令正式生效。</span></p>
<p><span style="font-family: 'Arial Unicode MS';">瑞典司法部长摩根?约翰逊（</span>Morgan Johansson<span style="font-family: 'Arial Unicode MS';">）表示</span>:<span style="font-family: 'Arial Unicode MS';">“希望该法起到阻慑作用，那些参与或提供恐怖主义相关训练的人会被判处最高两年监禁。如果我们尽早做好准备，就能够预防和避免一些犯罪。因此，我们要利用法律和预防工具，确保年轻人不会卷入这类犯罪。”</span></p>
<p><strong><span style="font-family: 'Arial Unicode MS';">恐怖份子普遍在本地有犯罪前科</span></strong></p>
<p><span style="font-family: 'Arial Unicode MS';">国家安全局情报处的斯塔夫</span>(Linda Staaf<span style="font-family: 'Arial Unicode MS';">）说：“我们可以看到，参与（恐怖）犯罪网络的人，在前往冲突地区参战之前，也在瑞典作案。”</span></p>
<p><span style="font-family: 'Arial Unicode MS';">斯塔夫谈到，这些人主要参与毒品犯罪、暴力犯罪、枪支犯罪等，同时对不同地区的恐怖份子提供资金。</span></p>
<p><span style="font-family: 'Arial Unicode MS';">在过去的一年中，安全局和警方增加了合作。斯塔夫表示，安全局正在和将要开展一些调查，一些罪犯也正在等待被开庭定罪。</span></p>
<p><span style="font-family: 'Arial Unicode MS';">斯塔夫认为，当局不仅要注重嫌疑人的恐怖犯罪活动，也要重视对其它罪行的监控。安全局负责人图恩伯伊（</span>Anders Thornberg<span style="font-family: 'Arial Unicode MS';">）持同样观点。他说</span>:<span style="font-family: 'Arial Unicode MS';">“这些罪犯通常既犯普通罪又犯恐怖罪。恐怖主义犯罪是最严重的，但你可以对其它犯罪行为釆取措施，这一点很重要。”他还表示，现在越来越多地釆用打击组织犯罪的方法来对付恐怖犯罪。</span></p>
<p><strong><span style="font-family: 'Arial Unicode MS';">注重对街头犯罪的调查</span></strong></p>
<p><span style="font-family: 'Arial Unicode MS';">图恩伯伊说，注重对街头犯罪的调查也是打击恐怖主义的一种方式。</span></p>
<p><span style="font-family: 'Arial Unicode MS';">“如果他们犯普通罪，就可以起诉他们，从而让这些人离开恐怖活动的舞台。”</span></p>
<p><span style="font-family: 'Arial Unicode MS';">禁止资助恐怖活动或送重礼</span></p>
<p><span style="font-family: 'Arial Unicode MS';">新法还明确规定，禁止资助那些犯下恐怖主义罪行的组织。不管资助目的是什么，只要是资助恐怖旅行或资助恐怖犯罪的个人或群体，就都是犯罪。司法部长约翰逊表示，新法规也留有余地，对不太贵重的礼物不予追究。他强调：从事恐怖主义活动的组织都有一个资金渠道，政府必须能够控制资金的流动。</span></p>
<p><strong><span style="font-family: 'Arial Unicode MS';">允许警方启动监测和窃听</span></strong></p>
<p><span style="font-family: 'Arial Unicode MS';">新法允许警方启动监测和窃听等方式对恐怖活动进行调查。约翰逊表示，有证人证词或其它情报显示嫌疑人有问题，就可以对其进行调查。</span></p>
<p><span style="font-family: 'Arial Unicode MS';">约翰逊说：“许多市民其实希望警察能够及时干预。某种形式的惩罚是必要的，因为这是严重的罪行。他们在恐怖活动现场犯罪，回到瑞典后，也会带来严重的安全隐患，因为他们接触的就是那类人（恐怖份子）。”</span></p>
<p><strong><span style="font-family: 'Arial Unicode MS';">期待更严措施</span></strong></p>
<p><span style="font-family: 'Arial Unicode MS';">反恐专家布朗（</span>Hans Brun<span style="font-family: 'Arial Unicode MS';">）说</span>:<span style="font-family: 'Arial Unicode MS';">“法案不会解决所有的问题，但它可以对恐怖份子的活动余地有大的影响，这是一个大的好消息。”</span></p>
<p><span style="font-family: 'Arial Unicode MS';">布朗认为立法是必要的，是正确道路，不是临时行为而是基于理念并有相应措施和动力支持。他呼吁建立一个广泛阵线，一个更强大的国际合作，以积极措施消除恐怖主义滋生的土壤，断绝巴尔干地区非法武器和制造炸弹所用化学品的供应。</span></p>
<p><span style="font-family: 'Arial Unicode MS';">瑞典各界也普遍认为，仅靠这一法案来消除恐怖主义还不够，需要配以其它更严格的紧缩措施。比如，从</span>4<span style="font-family: 'Arial Unicode MS';">月</span>15<span style="font-family: 'Arial Unicode MS';">日起，瑞典人</span></p>
<p><span style="font-family: 'Arial Unicode MS';">再可以随意申请护照，</span>5<span style="font-family: 'Arial Unicode MS';">年时间内最多可申请</span>3<span style="font-family: 'Arial Unicode MS';">次。瑞典司法部还将出具一份备忘录，以便瑞典在紧急情况下可从其它北欧国家获得支持。</span></p>
