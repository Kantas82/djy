<tr><td VALIGN=TOP width="626"><h2 align=center>瑞典五大工会会员将涨工资2.2%</h2>

<h6></h6>
<hr>
	<p>【大纪元2016年04月12日讯】（大纪元记者程雯编译报导）<span style="font-family: 'Arial Unicode MS';">经过与 雇主联合会的激烈对峙，瑞典工会、林木业印刷业工会、金属业工会、食品业工会和工程师工会组成的五大产业工会联合会在</span>2016<span style="font-family: 'Arial Unicode MS';">年涨工资谈判中获得胜利，这些工会的会员今年工资涨幅将达到</span>2.2%<span style="font-family: 'Arial Unicode MS';">。</span></p>
<p><span style="font-family: 'Arial Unicode MS';">经过激烈谈判，瑞典五大产业工会联合会（</span>FI<span style="font-family: 'Arial Unicode MS';">）与 雇主们就</span>2016<span style="font-family: 'Arial Unicode MS';">年的涨工资协议的谈判终于在</span>3<span style="font-family: 'Arial Unicode MS';">月</span>31<span style="font-family: 'Arial Unicode MS';">日深夜结束，在这五大产业中工作的大约</span>120<span style="font-family: 'Arial Unicode MS';">万名工会会员今年的工资将提高</span>2.2%<span style="font-family: 'Arial Unicode MS';">。这个结果让工会方面非常满意。</span></p>
<p><strong><span style="font-family: 'Arial Unicode MS';">双方开条件</span></strong></p>
<p><span style="font-family: 'Arial Unicode MS';">瑞典五大产业工会联合会包括瑞典工会、林木业印刷业工会、金属业工会、食品业工会和工程师工会。他们早前向 雇主联合会提出的</span>2016<span style="font-family: 'Arial Unicode MS';">年涨工资要求是一年涨</span>2.8%<span style="font-family: 'Arial Unicode MS';">，而 雇主方给出的涨工资幅度是两年涨</span>3.5%<span style="font-family: 'Arial Unicode MS';">。双方</span>2015<span style="font-family: 'Arial Unicode MS';">年度的工资协议于</span>3<span style="font-family: 'Arial Unicode MS';">月</span>31<span style="font-family: 'Arial Unicode MS';">日到期。</span></p>
<p><strong><span style="font-family: 'Arial Unicode MS';">工会获胜 雇主失望</span></strong></p>
<p><span style="font-family: 'Arial Unicode MS';">对于</span>2.2%<span style="font-family: 'Arial Unicode MS';">这个最后结果，工会方面非常满意。金属业工会主席安德斯?菲北（</span>Anders Ferbe<span style="font-family: 'Arial Unicode MS';">）满脸的喜悦，他认为这个协议结果让瑞典的劳动力市场继续保持了瑞典的行业标准。而 雇主方则非常失望。瑞典企业协会副总裁皮特?耶波松（</span>Peter Jeppsson<span style="font-family: 'Arial Unicode MS';">）认为，这个协议太昂贵了，这会让瑞典公司在全球竞争中越来越难。</span></p>
<p><strong><span style="font-family: 'Arial Unicode MS';">瑞典模式</span></strong></p>
<p><span style="font-family: 'Arial Unicode MS';">尽管 雇主方非常不情愿，但是双方最后能够达成协议，这是大家都期望的结果，人们认为这是遵循了瑞典模式，即通过谈判、妥协、谅解和共识，本着找到解决方法的态度最后达成协议。国家机构“调解研究所”派出的调解员也在双方谈判过程中起到了积极的作用，这是政府行为在解决劳资双方冲突中能够发挥作用的仅有的角色。</span></p>
<p><strong><span style="font-family: 'Arial Unicode MS';">其它工会受鼓舞</span></strong></p>
<p><span style="font-family: 'Arial Unicode MS';">五大产业工会联合会在涨工资谈判中的获胜，让其它工会也大受鼓舞。建筑业工会、交通运输业工会和商贸业工会与 雇主们的上年度工资协议已于</span>3<span style="font-family: 'Arial Unicode MS';">月</span>31<span style="font-family: 'Arial Unicode MS';">日到期，酒店餐饮业工会和市政工人工会的上年度工资协议将于</span>4<span style="font-family: 'Arial Unicode MS';">月底到期。建筑业工会主席约翰?林德霍尔姆（</span>Johan Lindholm<span style="font-family: 'Arial Unicode MS';">）坚持要求</span>3.2%<span style="font-family: 'Arial Unicode MS';">的工资涨幅，如果 雇主不接受，他们将从</span>4<span style="font-family: 'Arial Unicode MS';">月</span>12<span style="font-family: 'Arial Unicode MS';">日起罢工。如果 雇主不答应涨工资要求，油漆工工会也打算从</span>4<span style="font-family: 'Arial Unicode MS';">月</span>8<span style="font-family: 'Arial Unicode MS';">日起罢工。</span></p>
<p><strong><span style="font-family: 'Arial Unicode MS';">罢工谁损失</span></strong></p>
<p><span style="font-family: 'Arial Unicode MS';">如果工会号召罢工，工会将给参加罢工的会员发工资，这个工资的来源是从会员费中拨出的罢工储备金。因此工人参加工会罢工，不会对自己的经济有太大损失，但是没有人干活儿了， 雇主的损失将会是不可估量的。瑞典各大工会加起来总的罢工储备金达到</span>250<span style="font-family: 'Arial Unicode MS';">多亿瑞典克朗，而所有 雇主联合会加起来的应对基金则有大约</span>150<span style="font-family: 'Arial Unicode MS';">亿瑞典克朗。现在的瑞典社会，很少有罢工真正发生，但是在与 雇主谈判时提出罢工，会成为工会的一个非常有效的杀手镧。</span></p>

<p><strong><span style="font-family: 'Arial Unicode MS';">瑞典工会组织简介</span></strong></p>
<p><span style="font-family: 'Arial Unicode MS';">瑞典最大的工会组织是全国总工会（</span>LO<span style="font-family: 'Arial Unicode MS';">），它是</span>14<span style="font-family: 'Arial Unicode MS';">个行业工会的联合总工会，共有大约</span>150<span style="font-family: 'Arial Unicode MS';">万会员。这</span>14<span style="font-family: 'Arial Unicode MS';">个行业工会是建筑业工会、电工工会、房产物业工会、林木业印刷业工会、商贸业工会、酒店餐饮业工会、金属业工会（包括钢铁、化工、工程、建筑材料、塑料和制药等行业）、市政工人工会、食品业工会、音乐家工会、油漆工工会、造纸业工会、服务通信业工会和交通运输业工会。这些工会基本上是传统的蓝领工人工会组织。</span></p>
<p><span style="font-family: 'Arial Unicode MS';">对于受过高等教育的 雇员的工会组织是瑞典学者总工会（</span>Saco<span style="font-family: 'Arial Unicode MS';">），其中包括</span>23<span style="font-family: 'Arial Unicode MS';">个不同行业工会，如医生工会、建筑师工会、工程师工会、自然科学家工会、大学教师工会、中小学教师工会，经济师工会、律师工会，船长工会、药剂师工会，等等，共有会员</span>6<span style="font-family: 'Arial Unicode MS';">万</span>5<span style="font-family: 'Arial Unicode MS';">千名。</span></p>
<p><span style="font-family: 'Arial Unicode MS';">此外，国家公务员工会（</span>ST<span style="font-family: 'Arial Unicode MS';">）是面向政府机构 雇员的工会组织，有会员</span>9<span style="font-family: 'Arial Unicode MS';">万</span>3<span style="font-family: 'Arial Unicode MS';">千多名。瑞典工会（</span>Unionen<span style="font-family: 'Arial Unicode MS';">）既面向国家公务员，也面向私企员工，有会员</span>6<span style="font-family: 'Arial Unicode MS';">万</span>2<span style="font-family: 'Arial Unicode MS';">千名。</span></p>
<p><span style="font-family: 'Arial Unicode MS';">有些不同的行业工会又组合成工会联合会，以壮大声势，如上文中提到的五大产业工会联合会（</span>FI<span style="font-family: 'Arial Unicode MS';">）里有三个行业工会来自全国总工会，一个行业工会来自学者总工会，还有一个是瑞典工会。</span></p>
<p><span style="font-family: 'Arial Unicode MS';">全国总工会里面的</span>6<span style="font-family: 'Arial Unicode MS';">个行业工会，建筑业工会、电工工会、房产物业工会、油漆工工会、服务通信业工会和交通行业工会，也组合起来成立了一个工会联合会，称为</span>6F<span style="font-family: 'Arial Unicode MS';">。后来交通行业工会退出，但</span>6F<span style="font-family: 'Arial Unicode MS';">的名称继续沿用至今。</span></p>
