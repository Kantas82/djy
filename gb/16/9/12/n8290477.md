<tr><td VALIGN=TOP width="626"><h2 align=center>你是三星Note 7用户吗？ 赶快看这篇报导</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2016/09/note-7-600x400.jpg" />
<h6>AT＆T、T-Mobile和Verizon已停止提供更换三星Note 7服务，而三星也宣布停售Note 7，并呼吁用户停止使用该款手机。（视频截图） 
</h6>
<hr>
	<p>【大纪元2016年09月12日讯】（大纪元记者凌妃编译报导）Galaxy Note 7自上个月2日发布以来，有越来越多用户的手机引发火灾，美国消费者产品安全委员会（CPSC）和三星（Samsung）在9月9日正式向所有的Note 7用户发出警告，要求马上停止使用该款手机，并且关机。如果你是Galaxy Note 7的用户，一定要看这篇报导。</p>
<p>美国消费者产品安全委员会表示，某些Galaxy Note 7的“锂离子”（lithium-ion）电池在充电时，甚至是正常使用下，会引起火灾的疑虑，所以，警告所有Galaxy Note 7的用户应立即关机。</p>
<p>Galaxy Note 7的用户可直接拨打1-800-Samsung的电话，或是和所属的电信公司联系，以便更换替代手机或做后续处置。</p>
<p>当用户在拨打电话处理Note 7后续更换时，需要点耐心，已经有些Twitter的用户抱怨电信公司并没有足够的替换手机。</p>
<p>本文参考科技网站pcmag.com的报导，整理出美国各家电信公司针对Galaxy Note 7用户的后续处理程序建议。</p>
<h4>Verizon</h4>
<p>即日起至9月30日，Verizon免费为Galaxy Note 7用户更换替代手机。用户只要到Verizon商店或是打1-800-922-0204免费电话即可处理。</p>
<p><strong>AT&amp;T</strong></p>
<p>AT&amp;T力劝用户马上关闭Galaxy Note 7，并到当初购买手机的AT&amp;T店面更换手机。更换时用户要记得携带手机、充电器、配件、包装以及当初购买的收据。</p>
<p>如果用户是透过线上购买Galaxy Note 7，AT&amp;T会要求用户到店面更换手机。如果无法前往店面更换手机，需打电话至800-331-0500询问后续。</p>
<p>AT&amp;T表示，可以更换手机的SKU号码为6470A、6467A或是6469A。用户可以在手机包装盒上查到SKU号码。</p>
<p>AT&amp;T说：“Galaxy Note 7的用户可以选择Galaxy S7或是Galaxy S7 Edge作为取代机种，用户也可以退还Note 7的任何配件。如果用户决定其它的<ahref="https://github.com/eahkqg304/djy/blob/master/gb/tag/%E4%B8%89%E6%98%9F%E6%89%8B%E6%9C%BA.md#1">三星手机</a>，还有25美元礼券。”</p>
<p>当初因为购买Galaxy Note 7，而以优惠价购买Gear S2智能手表或是Galaxy Tab E平板电脑的用户，还是可以持续享有这些装置。如果不想拥有这些产品的消费者，AT&amp;T接受用户退货。</p>
<h4>T-Mobile</h4>
<p>在T-Mobile购买Galaxy Note 7的用户，可以取得全额退费，该笔款项可用于购买另一支替代手机。</p>
<p>用户可以直接到T-Mobile店面更换，或是拨打1-844-275-9309电话进行更换，T-Mobile会在下一个工作日将替代机邮寄给用户。</p>
<p>T-Mobile还提到，那些不想更换Note 7的忠实粉丝，可以等到三星的Note 7s有库存之后，马上更换，关于此的更多资讯，可以上T-Mobile的官网查询。</p>
<h4>Sprint</h4>
<p>Sprint的用户可以到店面更换Note 7手机，或是拨打1-888-211-4727电话询问更多的详细资料。假如用户选择以其它的<ahref="https://github.com/eahkqg304/djy/blob/master/gb/tag/%E4%B8%89%E6%98%9F%E6%89%8B%E6%9C%BA.md#1">三星手机</a>替代Note 7，可获得25美元礼券。</p>
