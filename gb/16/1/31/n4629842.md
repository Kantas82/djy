<tr><td VALIGN=TOP width="626"><h2 align=center>这5个Windows 10技巧让你工作更有效率</h2>

<h6></h6>
<hr>
	<p>【大纪元2016年01月31日讯】（大纪元记者凌妃综合报导）自2015年7月29日微软（Microsoft）开放用户免费下载Windows 10以来，目前全球已有超过2亿台电脑与装置安装新操作系统。有些Windows 10的操作技巧很实用，你可能还不知道？本文告诉你，让你操作更快速、工作更有效率。</p>
<p><h2>1. 利用键盘上的Win键快速开启“开始功能表”</h2>
<p>Windows 10又回到了“开始功能表”，类似Windows 7的传统操作界面，比起Windows 8满满都是“瓷砖”的桌面更容易作业。但是，Windows 10并没有完全放弃“瓷砖”的开始界面，而是将它与开始功能表结合在一起。</p>
<p>开启键盘上的【Win】（视窗）按键后，利用键盘的方向键就能快速地选择自己需要的工作项目，并快速启动。</p>
<p><h2>2. 快速找出常用与新增的App</h2>
<p>在 Windows 10的“开始功能表”中，会自动显示出【最常使用】与【最近新增】的 App、软件或是功能，只要善加使用这份清单，就可以有效率地寻找资讯。</p>
<p><h2>3. 自订个人专属的工作平台</h2>
<p>用户可透过Windows 10 的“开始功能表”自订个人专属的工作平台，排序出最常用的工具，以迅速开启常用的资料夹或系统设定。</p>
<p>打开“开始功能表”右边的自订工作版面，到应用程序清单上点击右键，然后选择【钉选到开始画面】，就可以把常用的App钉选到右半边的开始画面。</p>
<p>用户可以利用“开始功能表”右边的自订工作版面做分类或排序，例如：设计“代办事项”或是“即时新闻”等，然后将相关的软件拉入板块即可。</p>
<p><h2>4. 让“开始功能表”更美观</h2>
<p>在桌面上点击鼠标右键选择【个人化】，选择【色彩】，这时选项代表的意义是：<br />【自动从我的背景选择辅色】：开始画面中的动态瓷砖颜色会根据你的桌布自动变换。<br />【在开始功能表显示色彩】：“开始功能表”背景颜色会根据桌布颜色变换，如果取消这项功能，“开始功能表”就会显示为黑色。 <br />【将开始功能表变成透明】：“开始功能表”会显现出雾面玻璃的效果。</p>
<p><h2>5. 虚拟桌面的快捷键</h2>
<p>Windows 10的一个新功能就是“虚拟桌面”（Virtual desktops），而虚拟桌面有一些快捷键很实用。它们是：<br />【Windows Key + Ctrl + D】：建立一个新的虚拟桌面。<br />【Windows key + I】：开启设定画面 。<br />【Windows Key + Ctrl + F4】：关闭正在浏览的虚拟桌面。<br />【Windows Key + Ctrl + 方向键Left】：将虚拟桌面向左移。<br />【Windows Key + Ctrl + 方向键Right】：将虚拟桌面向右移。 <br />【Windows Key + Tab】：视窗快速切换。 </p>
