<tr><td VALIGN=TOP width="626"><h2 align=center>工作与生活条件 欧洲囊括前十名 丹麦第一</h2>

<h6></h6>
<hr>
	<p>【大纪元2012年01月11日讯】（大纪元记者唐喜梅编译报导）<ahref="https://github.com/hcmkav320/djy/blob/master/gb/tag/%E6%8D%B7%E5%85%8B.md#1">捷克</a>官方媒体《捷克新闻》（Novinky.cz）2012年1月10日报导，在社会保险、工作就业等工作与生活条件比较排名上，北欧占优势。世界<ahref="https://github.com/hcmkav320/djy/blob/master/gb/tag/%E7%BB%8F%E6%B5%8E.md#1">经济</a>合作发展组织（OECOD）1月份对2011 年的调研中，考虑到了其中带孩子就业母亲所占的百分比；人们每周的自由时间和大量超时工作人员的百分比因素，丹麦名列世界第一，捷克共和国世界排名第13位。</p>
<p>北欧国家，如丹麦、瑞典和挪威，在工作和生活方面的优越性均位于世界前七名，北欧国家在名次排列中占优势，几乎占居了前10名。此外，世界经合组织认为，丹麦是最缺少儿童的国家。</p>
<p>在美国，虽然带孩子的母亲就业百分比例高，但是相当多的人在就业中负担过重，而且与北欧国家相比，人们的自由时间少。</p>
<p>根据世界<ahref="https://github.com/hcmkav320/djy/blob/master/gb/tag/%E7%BB%8F%E6%B5%8E.md#1">经济</a>合作发展组织报告，2011年度工作与生活条件最好的国家排名如下：</p>
<p>国家排名超时工作人员带孩子的就业母亲每天自由时间<br />1. 丹麦：0,02 % / 78 % / 16,31 hod.<br />2. 挪威：0,03 %/79 % / 16,05 hod.<br />3. 荷兰：0,01 %/76 % / 16,06 hod. <br />4.芬兰： 0,04 % / 76 % / 15,95 hod. <br />5. 比利时： 0,04 % / 63 %/ 16,61 hod.<br />6. 瑞士：0,06 % /79 %/15,74 hod.<br />7. 瑞典： 0,01 % / 76 %/15,48 hod.<br />8. 德国： 0,05 % / 66 %/ 16,14 hod. <br />9. 葡萄牙： 0,06 % / 65 % / 15,27 hod. <br />10. 法国：0,09 %/66 % / 16,06 hod. <br />11. 斯洛维尼亚： 0,07 % /74 % /15,29 hod. <br />12. 爱沙尼亚： 0,06 %/ 65 %/ 15,27 hod. <br />13. <ahref="https://github.com/hcmkav320/djy/blob/master/gb/tag/%E6%8D%B7%E5%85%8B.md#1">捷克</a>： 0,09 % /71 % / 15,48 hod. <br />14. 加拿大： 0, 06%/ 65 % /15,27 hod. <br />15. 冰岛：0,14 % /87%/14,29 hod. <br />16. 卢森堡： 0,04 %/ 57 % /15,57 hod. <br />17. 英国： 0,12 %/ 67 % / 15,60 hod. <br />18. 奥地利： 0,10 % /71 %/ 15,23 hod. <br />19. 斯洛伐克: 0,06 % /65 % / 15,27 hod. <br />20. 匈牙利: 0,03 % /59 %/15,93 hod. <br />21. 希腊:0,06 %/ 52 %/ 15,86 hod. <br />22. 西班牙: 0,07 %/ 57 % / 15,70 hod. <br />23. 美国:0,11 %/ 73 %/15,13 hod.</p>
<p>*超时工作系指工作每周超过50小时</p>
