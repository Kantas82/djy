<tr><td VALIGN=TOP width="626"><h2 align=center>【细数费城大学篇之二】天普大学(Temple  University)</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2009/03/903201053542088-589x400.jpg" />
<h6>图﹕天普大学位于费城。图为黑人影星考斯比在天普大学演讲。(Getty Images)
</h6>
<hr>
	<p>【大纪元3月20日讯】上一期我们和大家介绍了宾州州立<ahref="https://github.com/qespzd3957/djy/blob/master/gb/tag/%E5%A4%A7%E5%AD%A6.md#1">大学</a>，今天我们继续向大家介绍宾州三所公立大学之一的<ahref="https://github.com/qespzd3957/djy/blob/master/gb/tag/%E5%A4%A9%E6%99%AE%E5%A4%A7%E5%AD%A6.md#1">天普大学</a>。可能很多人对于天普大学都是耳熟能详的，那么您知道这所大学成立的故事吗？让我们从100多年前开始……<br />
100多年前，费城的6个高中生向他们仰慕已久的一位博学多才的牧师请求：“先生，您肯教我们读书嘛？我们想上<ahref="https://github.com/qespzd3957/djy/blob/master/gb/tag/%E5%A4%A7%E5%AD%A6.md#1">大学</a>，可是我们没钱。我们中学快毕业了，有一定的学识，你肯教我们嘛？”</p>
<p>这位牧师名叫康惠尔，他答应教这6个贫家子弟，同时他又暗自思忖：“一定还会有许多年轻人没钱上大学，他们想学习但付不起学费。我应该为这样的年轻人办一所大学。”</p>
<p>于是，他开始为筹建大学募捐。当时﹐建一所大学大概要花150万美元。</p>
<p>康惠尔四处奔走，在各地演讲了5年，恳求有志于学的年轻人捐钱。出乎他意料的是，5年辛苦募捐到的钱不足1,000美元。康惠尔深感悲伤，情绪低落。当他走向教堂准备做礼拜的演说词时，低头沉思的他突然发现教堂周围的草枯黄得东倒西歪。他便问园丁：“为什么这里的草不如别的教堂周围的草呢？”</p>
<p>园丁抬起头来望着牧师回答说：“噢，我猜想你眼中觉得这地方的草长得不好，主要是因为你把这些草和别的草相比较的缘故。看来，我们是常常看到别人美丽的草地，希望别人的草地就是我们自己的，却很少去整治自家的草地。”园丁的一席话使康惠尔恍然大悟。他跑进教堂开始撰写演讲稿。</p>
<p>他在演讲中讲了一个农夫的故事：<br />
有个农夫拥有一块土地，生活过得很不错。但是，当他听说可以找到埋有钻石的地方时，他便想﹐只要有一块钻石就可以富得难以想象。于是，农夫把自己的地卖了，离家出走，四处寻找可以发现钻石的地方。农夫走向遥远的异乡他国，然而从未能发现钻石，最后，他囊空如洗。有一天晚上，他终于在海滩自杀身亡。</p>
<p>真是无巧不成书。那个买下这个农夫的土地的人，在散步中无意间发现了一块异样的石头，拾起一看，它晶光闪闪，反射出光芒。仔细察看，发现这是一块钻石。这样，就在农夫卖掉的这块土地上，新主人发现了从未被人发现的钻石宝藏。</p>
<p>这个故事是发人深省的，康惠尔写道：财富不是仅凭奔走四方去发现的，它属于自己去挖掘的人，属于依靠自己土地的人，属于相信自己能在自己的土地上创造奇迹的人。康惠尔做了7年这个“钻石宝藏”的演讲。7年后，他筹得800万美元，这笔钱大大超出了他所想建一所学校的需要。</p>
<p>今天，这所学校竖立在宾夕法尼亚州的费城，这便是著名学府——<ahref="https://github.com/qespzd3957/djy/blob/master/gb/tag/%E5%A4%A9%E6%99%AE%E5%A4%A7%E5%AD%A6.md#1">天普大学</a>。</p>
<p>天普大学建于1884年，是一所拥有34,000名本科生﹑研究生和专业职业学生的综合性研究大学，规模居全美第27。大学内设十七所学院，包括商学院、医学院、药学院、和牙医学院。</p>
<p>天普大学总共有五大校区，遍布全费城。天普最特殊的一点即是在美国本土之外，另有二个海外校区﹐一在意大利的罗马，一在日本的东京。因此﹐毫不意外的，天普有相当大的文化融合力。这一点由学生国籍的多样化中表现得一览无遗。</p>
<p>据《耶鲁大学日报大学指南》指出，天普大学设有一些核心课程，学生不论修读任何学系或学科，均须进修写作、数学、科学﹑社会学、历史、美术、美国文化及外文等学科。大学最负盛名的学科是电台／电视／电影学，以及舞蹈和音乐。学生修读最多的学科，依次是商科管理（28%）、大众传播（13%）、教育（9%）、社会科学（8%）和工程（7%）。本校以电影 ( 全美排名第七 ) ，大众传播 ( 第十四 ) ，人文艺术( 第十四 ) ，理疗 ( 第十七 ) ，教育 ( 第二十五 ) ，社工 ( 第二十一 ) ，法律 ( 第五十一 ) ， MBA( 第六十四 ) 等排名为高。值得一提的是，法学院的法庭诉讼组排名全美第一；医学院心脏移植全美颇负盛名。而人数较多的科系，有MBA, Mass Media, Law, CIS, Music, Dance, Medical School 等。</p>
<p>天普大学共设有300个学位项目，其中准学士学位项目2个（园艺学与通用学科），学士学位项目125个，硕士项目113个，博士项目52个，以及在牙科、法学、医学、药学以及足病学领域的8个第一专业学位项目。另外还有10个证书项目。</p>
