<tr><td VALIGN=TOP width="626"><h2 align=center>Orion String Quartet四重奏  东吴登场</h2>

<h6></h6>
<hr>
<p>【大纪元3月24日报导】（中央社记者林思宇台北24日电）长期在纽约林肯中心驻团的Orion String Quartet弦乐四重奏团，应东吴大学邀请来台交流，明晚起举行3场<ahref="https://github.com/eydeib3500/djy/blob/master/gb/tag/%E9%9F%B3%E4%B9%90%E4%BC%9A.md#1">音乐会</a>，由于民众索票踊跃，未拿到票的民众可参与27日的彩排活动。</p> <p>东吴表示，Orion String Quartet将进行为期一周的国际学术交流活动，举办5场大师讲座并分别在25 、26、27日举行3场<ahref="https://github.com/eydeib3500/djy/blob/master/gb/tag/%E9%9F%B3%E4%B9%90%E4%BC%9A.md#1">音乐会</a>；音乐会门票已被索取一空，主办单位为不让听众失望，特别开放民众参与周五上午10时至下午1时的彩排。</p> <p>东吴指出，Orion String Quartet时常在北美、欧洲表演，2008年为KOCH International Classcis录制整套贝多芬的弦乐四重奏，获得乐界极高评价。</p> <p>东吴说，Orion String Quartet此行为首度亚洲之旅，第一站就选择在台北外双溪的东吴大学。</p>
