<tr><td VALIGN=TOP width="626"><h2 align=center>汉翔浮报军机维修费案 起诉7人</h2>

<h6></h6>
<hr>
<p>【大纪元7月20日报导】（中央社记者张荣祥台南市20日电）汉翔公司疑勾结浮报军机<ahref="https://github.com/mjerqg3317/djy/blob/master/gb/tag/%E7%BB%B4%E4%BF%AE%E8%B4%B9.md#1">维修费</a>新台币1亿4000多万元案，台南地检署今天侦结，起诉汉翔专案组长李宗政等7人。</p>
<p>起诉书指出，国防部军备局采购中心民国 94 年10月间办理COCO专案，将空军第2后勤指挥部所属军工厂委托民营，且公开评选招标，最后由汉翔公司联合汉翔管理顾问公司、亚洲航空、长荣航太科技，共同以71亿余元，承揽委托经营8年，负责UH-1H等陆海空军军机的维修。</p>
<p>非合约厂商、前美国 API 亚太地区经理张平，为打进 COCO 专案，取得汉翔公司的订单，事前刻意和李宗政结交，且自民国 94 年 4 月涉嫌以每个月 1000美元行贿李宗政，后因感谢李宗政协助亚航取得大量军方交修委托案，每个月行贿金额增至 1200 美元。</p>
<p>张平和前亚航总经理于羾两人更分别代表非合约厂商美国 API 公司及合约厂商亚航，签订合作互惠协议书，约定双方交易后的浮报利润，以 6:4 共分。</p>
<p>于羾又涉嫌透过时任亚航专员的赵高恩，和非合约厂商凌威公司总经理李荣裕接洽，将凌威拥有的维修军机能量清册，浮报在亚航的能量清册中，让军方误以为亚航维修能量很高，才将大量待修军机转到李宗政负责的专案，由亚航维修。</p>
<p>因亚航无维修能量，于羾、赵高恩、亚航业务处处长崔人俊、物料处处长张兴南等人，再将这些待修件送到美国 API 公司或凌威公司维修，且加计维修成本浮报，造成国防部逾亿元的损失。</p>
<p>南检侦结后，以贪污、伪造文书、诈财、违反公务员服务法等罪嫌，起诉李宗政、赵高恩、张平、于羾、崔人俊、张兴南、李荣裕等7人。</p>
