<tr><td VALIGN=TOP width="626"><h2 align=center>家长孩子翘首期盼的双语幼儿园（下）</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2018/03/01_DSC_0155-600x400.jpg" />
<h6>Golden State Preschool幼儿园的学童，在中文课堂上学习中文字。（湾区双语幼儿园GSP提供）
</h6>
<hr>
<p>【大纪元2018年03月12日讯】（大纪元记者余亨之旧金山湾区报导）中半岛最大的中文课后班&#8211;Intelligent Learning（Golden State Preschool的前身），5年来深受家长和孩子的喜爱。在家长极力的要求下，为了给更多孩子提供优质的早幼儿双语教育，Golden State Preschool在南旧金山、圣马刁（San Mateo）筹备已久的两个分校，将于2月1日开学。许多妈妈闻讯赶来，其中还有怀胎5个月的准妈妈，前来为肚里的宝宝报名。上集谈到该校的教育理念与师资，本周能看到学校课程与家长喜爱的原因。</p>
<h2>湾区幼儿学校：Golden State Preschool对症下药，不能一味奖励</h2>
<p>校长Irina Yang认为，“赏识教育”不是对所有的孩子都管用。有的小孩完成了一件事情，你给他一个奖励，他不会因此走向良性循环。因为他完全不知道为什么要学习，他变成了为得到那个玩具而写作业；玩具拿到手后，他就把作业扔在一边了。</p>
<figure id="attachment_10210069" aria-describedby="caption-attachment-10210069" style="width: 300px" class="wp-caption aligncenter"><a target="_blank" href="https://i.epochtimes.com/assets/uploads/2018/03/02_Irina-Yang.jpeg"><img class="size-small wp-image-10210069" src="https://i.epochtimes.com/assets/uploads/2018/03/02_Irina-Yang-300x411.jpeg" alt="" width="300" b="411" /></a><figcaption id="caption-attachment-10210069" class="wp-caption-text">Golden State Preschool幼儿园的校长Irina Yang。<ahref="https://github.com/hcmkav320/djy/blob/master/gb/tag/%E5%8D%97%E6%97%A7%E9%87%91%E5%B1%B1%E5%B9%BC%E5%84%BF%E5%9B%AD.md#1">南旧金山幼儿园</a>GSP提供）</figcaption></figure>
<p>对于贪玩的男孩，Irina就建议妈妈让孩子学跆拳道、功夫课，孩子释放能量后就能坐得住了，能静下心来学习。有一个男孩上课时坐在凳子上，总是习惯把身体从一边扭到另一边，不停地扭，老师站在旁边也管不住。Irina就买了一个转盘，放在他的凳子上，用吸盘吸住；他坐在上边不停地扭动，扭到最后他自己把转盘拿走了，能安静地上课了。</p>
<h2><ahref="https://github.com/hcmkav320/djy/blob/master/gb/tag/%E4%B8%AD%E5%8D%8A%E5%B2%9B%E5%B9%BC%E5%84%BF%E5%9B%AD.md#1">中半岛幼儿园</a>：Golden State Preschool小孩学中文，潜移默化</h2>
<p>Golden State Preschool双语幼儿园分为三个班级：2～3.5岁班，3.5～4.5岁班，4.5～5.5岁班，中文教材是《美洲华语》。前两个班级，采用沉浸式国语教学，也教一些基本英语，主要以国语为主。课外活动有毛笔字、国画、手工等。</p>
<p>虽然上中文课，可美国长大的小孩彼此之间更喜欢用英语交流。有的妈妈见此情景很焦急，对孩子说：“你花钱上中文学校，不说中文，老说英文，不是白学了吗？”。一年暑假，这个妈妈带着两个孩子回了一趟大陆，两个孩子用中文与周边的人交流，妈妈感到非常惊讶，给杨老师发微信说：我的两个孩子竟然都能用国语跟人交流！杨老师说：我不会让他们每天写多少字拿回去给您看，也很难保证他们在美国的学校一直说中文。可是一旦到了中文的环境，他们就能说中文。</p>
<h2>“三岁看小，七岁看老”，好习惯更重要</h2>
<p>中国民间流传的一句古老谚语“三岁看小，七岁看老”：从三岁孩子的心理与行为的表现上，可看到他到青少年时期的心理与个性形象的雏形；从七岁孩子身上，能看到他中年以后的成就。</p>
<p>杨校长的教育理念与此不谋而合，她认为孩子在幼小的时候，保持一个好的道德规范、行为礼仪，如用餐礼仪、见老师有礼貌、关心他人……在心智方面对孩子的培养，比学知识更重要。</p>
<p>Golden State Preschool每天早上9：00至9：30的例程是：上洗手间，解决大小便、洗手、整理衣服，回到教室坐着等待老师，才开始吃点心。孩子在这里不仅养成良好的行为习惯，还学会关心他人，与人分享。“Share”、“Care”是孩子之间经常说的。</p>
<p>小孩每天活动量大，代谢旺盛，需要保证一定的饮水量。Golden State Preschool的小朋友每天到校第一件事，就是在饮水机前把自己的水壶装满水，放学回家前把水喝完，否则不让走。</p>
<p>许多小孩上课，喜欢把鞋子一扔，踢的到处都是。杨校长坚持要求孩子一定穿上鞋子，老师们会蹲下，手把手的教孩子系鞋带，直到他们学会为止。孩子会把好习惯带回家里，家里的鞋子也是摆得整整齐齐的。</p>
<h2><ahref="https://github.com/hcmkav320/djy/blob/master/gb/tag/%E4%B8%AD%E5%8D%8A%E5%B2%9B%E8%AF%BE%E5%90%8E%E7%8F%AD.md#1">中半岛课后班</a>：Golden State Preschool一心为孩子，受到家长热捧</h2>
<p>Golden State Preschool为孩子提供有机午餐，从Wholefood或Trader Joe购买有机牛奶、蔬菜、果汁、肉类等；煮饭的阿姨备有健康证，个人背景调查资料。杨校长的教师团队非常关注孩子的身心健康，因此收到家长们的热捧。</p>
<p>Intelligent Learning的夏令营去年没做任何广告，报名人数从第一个星期的20人，到接下来的每个星期都是50人。夏令营课程安排的非常有趣，有首饰（Jewelry）、拼图（Puzzle）、3D Printer、编码（Coding）等，孩子过的开心，家长口耳相传，以至每星期的报名人数爆满。</p>
<p>Intelligent Learning在Millbrae、San Mateo、Burlingame已有三家分校，每家分校早已满额。现在Golden State Preschool南旧金山市分校，将于2月1日开学。周边的家长纷纷慕名前来，其中一个孕妇手里拉着一个小朋友，跟杨老师说要报两个名额，包括肚里的孩子。</p>
<p>这么好的双语幼儿园，家长怎能不翘首以盼呢！如果您正在为孩子选择幼儿园而烦恼，不妨去体验一下吧！每个班级名额有限，预约排位请从速！</p>
