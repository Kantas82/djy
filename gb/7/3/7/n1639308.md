<tr><td VALIGN=TOP width="626"><h2 align=center>Web 2.0 / 3.0自由民主化的全球网路大潮</h2>

<h6></h6>
<hr>
	<p>【大纪元3月7日讯】(大纪元特约记者唐浩报导) 不管是翻开报纸、打开电视还是登上<ahref="https://github.com/hcslle3160/djy/blob/master/gb/tag/%E7%BD%91%E8%B7%AF.md#1">网路</a>，您是不是也发现大家都在谈 Web 2.0，甚至是 Web 3.0？这几个稀奇古怪的英文和数字符号的组合，究竟是什么意思呢？</p>
<p>而且，有趣的是，不少人去听了、看了许多所谓 Web 2.0的企业案例或者 Web 2.0的创业故事后，却似乎还是不太明白 Web 2.0到底是什么。本文马上就带您揭开 Web 2.0的神秘面纱。</p>
<p><B> Web 2.0的真面目</B></p>
<p>Web 2.0一词是由<ahref="https://github.com/hcslle3160/djy/blob/master/gb/tag/%E7%BD%91%E8%B7%AF.md#1">网路</a>趋势专家欧莱利（Tim O’Reilly）在2004年所提出的概念名词。简单地说，就是指网站具备某种特性，让服务提供端（provider）与用户端（client）两者之间能产生“互动”与“分享”的行为，而网站就作为双方、甚至是多方之间对等沟通的平台。在这种状态下，网站不再只是用户单方面取得资料与聆听讯息的管道，而是用户也可以自由地贡献内容并表达个人意见、特色的舞台。</p>
<p>台湾这类网站的早期先例，以2000年创立的《明日报》个人新闻台为代表。它让使用者藉由文字记录形式来撰写日记，分享个人的创意或心情写真。不过当时只提供用户分享文字创作，再搭配相当阳春的图片上传功能。</p>
<p>近几年最著名的 Web 2.0典型，莫过于由台湾交通大学学生开发的“无名小站”，它不仅延续文字创作分享的传统（即“部落格”或“博客”），并同时提供图片，甚至是影像、音乐的分享功能。而去年10月刚被 Google以16.5亿美元收购的 YouTube，不仅是当前全球最大的影片分享平台，也是经营 Web 2.0“由创意到获利”的传奇故事之一。</p>
<p><B>百花齐放、权力共享的民主风格 </B></p>
<p>只要曾经接触过这类网站便不难发现，Web 2.0并不复杂，它所呈现的风格是“共享”而非“独占”；是意见多元、百花齐放的“民主论坛”，而非一人独大、暮气沉沉的“一言堂”。几乎任何人都可以在这类网站上自由地发表言论、创作自己的作品、与他人分享自己的许多东西。</p>
<p>所以，Web 2.0并不是用什么酷炫的高科技来作为划分的标记，而单纯是一种网站经营观念的重大转变。藉由服务提供端的“权力下放”、“为使用者提供技术服务”，从而招徕无数的自由作家与自由工作者，共同为这块广大的开放平台进行多采多姿的义务耕耘。最终这些耕耘化成了具体营收，作为对业者提供服务的最佳回报。</p>
<p>套几个专业术语来说得深入一点。从资讯管理角度而言，Web 2.0是强化网际网路的“民主性”，强调“网路使用者经验”以及“网路使用者意识”的提升，并加强互动机制，提供层次更丰富、更深度的使用者体验及服务。让使用者不再只是被动地撷取网路资料，而是能积极地、主动地主导网站的内容编撰、甚至是提供全新内容（如知名的维基百科 Wikipedia就是个典型的范例），从而吸引广大的用户来为Web 2.0网站引入丰沛的资讯生产能量，创造出 Web 2.0的获利基点。</p>
<p>同时，网路使用者在 Web 2.0网站上找到挥洒自己能力（如部落格）、突出自我个性（如网络相簿）、标榜个人创意（如网路影音分享）的舞台，而这些舞台绝大多数都是免费的（至少表面上是如此），使用者藉由这种唾手可得的管道来表现自己、凸显自己，享受那份成名的愉悦、感受那份分享的温馨。</p>
<p>某种角度来说，Web 2.0藉由网路内容主导权的下放分享，取得网路服务提供者（provider）和使用者（user）的和谐亲近，双方也都各取所需、各有所得，创造出一片相辅相成、共荣共存的虚拟新天地。</p>
<p>当然，或许从某种角度来看，Web 2.0的提供者和用户似乎都是为名而来、为利而聚，是为了“利己”而形成的一种网路有机体。但这种观察并不充分，甚至是有些倒果为因。</p>
<p>倘若我们再深入窥探，Web 2.0其实是一种心灵上、精神上的“新生活运动”，它藉由网际网路作为载体，将服务提供者、使用者彼此的心胸拓开宽敞，解除自我封闭、敝帚自珍的狭小胸怀，创造了一种崭新的网路使用经验：共享、公有、利他、开放、集体创新、群体智慧等等。</p>
<p><B>彻底开放、高度民主化的 Web 3.0 </B></p>
<p>不过，有人对于 Web 2.0的网路服务内容（例如网页版型、连结架构等）还是得受制于服务端所提供的少数样式而感到不满，使用者的创意与能力还是遭到服务提供者某种程度的“拘禁”。</p>
<p>而且，对于服务提供者在当 Web 2.0网站获得一定数量的使用者青睐后，就擅自把网站连同使用者整批论斤论两地转卖给其他业者，更是引起不小的批判挞伐，认为辛勤耕耘网站内容的用户们并没有得到合理的尊重与对待。</p>
<p>台湾中正大学的吴昇教授等人因而提出了 Web 3.0的概念。套用美国林肯总统的“民有、民治、民享”的民主主张，他们认为 Web 2.0或许做到了“网友治”（by the user）、“网友享”（for the user），但并没有真正做到“网友有”（of the user）。整个网站还是归属服务端业者所拥有，众多的网友不过是业者的人力财产。</p>
<p>于是吴昇等人发展出一套新的网路系统：Nuweb，藉由这套系统，任何人在任何电脑上都可以轻松地制作网站、管理网站，自由地分享各种资讯。Nuweb包含一套服务端软体（给中央服务器使用）以及一套用户端软体，用户电脑里只要安装了这套软体，就等于在您的电脑里建立了自己的专属网站，只要一上网，中央服务器就会根据您的使用者名称来自动帮您转换IP（Internet Protocol，网路通讯协定）位址，就能透过中央服务器的转接，让世界各地的网友都能轻松地看到您的网页，取得您想要分享的文字、照片、音乐或其他档案。</p>
<p>Web 3.0这套系统目前还在积极地研发测试，虽然有人认为 Nuweb还看不到太大的突破与影响力，但是这些新态度与新思维，某种程度也正反映了民主意识更为进步、自由理念更为开放、平等意识更为落实的今日台湾社会。</p>
<p><B>科技民主浪潮的社会冲击 </B></p>
<p>值得注意的是，也正由于 Web 2.0、Web 3.0具备如此高的民主化特性、去权力中心化的特质，也使得一些害怕民主自由的极权国家对 Web 2.0进行封锁、排斥或抵制。中共政府就是一个例子，一些特定的 Web 2.0网站在中国大陆境内部分地区或甚至全国境内已经遭到封锁，中国大陆的网路用户并无法连线。</p>
<p>包括中共、伊朗、阿拉伯等采取网路封锁的国家，面对这股在全球各地风起云涌的Web 2.0 / 3.0浪潮，这种“愚人民以保权位”的统治手段，究竟能否抵挡自由化、民主化、去权威化的资讯洪潮与管理营运模式？或者还能抵挡多久？这股资讯科技的新生活运动，对于一个极权封闭的国家政权将会带来如何强大的冲击与影响？这一切，在不久的将来都将在我们眼前一幕幕上映。</p>
<p><B> Web 2.0让人人都能成为新闻记者</B></p>
<p>Web 2.0让使用者自行提供网页内容，文字与图像兼具的特性，吸引了无数的网路使用者乐在其中。因为几乎所有人都可以成为记者，把自己日常生活中所见、所闻的特殊事件加以记录，很可能就是篇独一无二的独家新闻。于是，Web 2.0悄悄带来一股新兴的新闻浪潮：网路公民新闻报导。</p>
<p>公民新闻报导（又称公共新闻报导，citizen/public journalism）最早在1998年发源于美国，主要是指新闻媒体透过民调、公听会等方式，来听取民众的意见，观察民众真正感兴趣的议题是什么、整个社区真正迫切需要的是什么，媒体再将这些意见作为参考来进行新闻采访与报导。这种将公众的意见纳入新闻报导的作法，等于是由媒体记者与读者一起来创作新闻，不仅让读者的意见获得政治人物的重视，也让读者更乐于阅读媒体、参与媒体新闻和社区公共事务。</p>
<p>Web 2.0让使用者人人可自由上传自己创作的文字、图片甚至音乐、影像等，让全球网友自由浏览，于是有部分网站灵机一动，推出“全民皆记者”的构想。人人都可以把自己创作的“新闻”上传到网站上，再由网站上的专业编辑来审核、修改后，就可以发表出来成为正式的新闻报导，在网站上供人点阅，而记者甚至还可以获得“稿费”做为奖励。</p>
<p>最著名的网路公民新闻网站，首推韩国的OhmyNews。另外，台湾的公共电视目前也正在筹画类似的服务，近期即将推出。</p>
